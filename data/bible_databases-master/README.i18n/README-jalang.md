# 聖書のバージョンと相互参照データベース: MySQL、SQLite、CSV、JSON、YAML、TXT、MD。

これは、さまざまな形式の聖書のバージョンを集めたものです。以下にいくつか簡単に紹介します。

- [Formats Folder](.././formats) で探している聖書のバージョンを見つけます。
- [Sources Folder](.././sources) でソース テキスト/リソースを検索します。
- [Scripts Folder](.././scripts) にあるこのプロジェクトの生成/変換スクリプトを利用します。
- 大規模で正確な相互参照リソースが https://www.openbible.info/labs/cross-references/ から含まれています。

単純な [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md) を参照してください。

> **⚠️ 重要: このプロジェクトのレガシー バージョンは、[2024](https://github.com/scrollmapper/bible_databases/tree/2024) ブランチで入手できます。データベース スキーマへの大幅な変更が 2025 ブランチ以降のバージョンに実装されていることに注意してください。**

## データベーススキーマ

次のセクションでは、このプロジェクトで使用される一般的なデータベース スキーマについて説明します。

#### テーブル: `<翻訳>_books`
この表には、指定された翻訳聖書のすべての書籍がリストされています。

|列名 |タイプ | Null 可能 |キー |デフォルト |番外編 |説明 |
|-----------|------|----------|---------------|-----------|-----|----------------------|
| `id` |整数 |いいえ |主キー | NULL |自動インクリメント |各書籍の一意の識別子。  |
| `名前` | varchar(255) |はい |             | NULL |                |本の名前。             |

#### テーブル: `<translation>_verses`
この表には、指定された翻訳聖書のすべての節が含まれています。

|列名 |タイプ | Null 可能 |キー |デフォルト |番外編 |説明 |
|-----------|------|----------|---------------|-----------|-----|----------------------|
| `id` |整数 |いいえ |主キー | NULL |自動インクリメント |各詩の一意の識別子。 |
| `書籍ID` |整数 |はい |インデックス | NULL |                |書籍の ID (`<translation>_books` への外部キー)。 |
| `章` |整数 |はい |             | NULL |                |章番号。               |
| `詩` |整数 |はい |             | NULL |                |詩番号。                 |
| `テキスト` |テキスト |はい |             | NULL |                |詩のテキスト。            |

#### 表: 「翻訳」
この表には、利用可能な聖書翻訳に関する情報が含まれています。

|列名 |タイプ | Null 可能 |キー |デフォルト |番外編 |説明 |
|--------------|--------------|----------|---------------|-----------|----------|----------------------------|
| `翻訳`| varchar(255) |いいえ |主キー | NULL |       |翻訳の略称。 |
| `タイトル` | varchar(255) |はい |             | NULL |       |翻訳の完全なタイトル。 |
| `ライセンス` |テキスト |はい |             | NULL |       |翻訳のライセンス情報。 |

#### テーブル: `cross_references`
この表には、異なる節間の相互参照データが含まれています。

|列名 |タイプ | Null 可能 |キー |デフォルト |番外編 |説明 |
|--------------|--------------|----------|---------------|-----------|-----|--------------------------------------|
| `id` |整数 |いいえ |主キー | NULL |自動インクリメント |各相互参照エントリの一意の識別子。 |
| `本から` | varchar(255) |はい |インデックス | NULL |                |相互参照が開始される本。 |
| `からの章` |整数 |はい |             | NULL |                | 「from_book」の章番号。 |
| `from_verse` |整数 |はい |             | NULL |                | 「from_book」内の節番号。 |
| `to_book` | varchar(255) |はい |インデックス | NULL |                |相互参照が指す本。 |
| `to_章` |整数 |はい |             | NULL |                | 「to_book」の章番号。 |
| `to_verse_start`|整数 |はい |             | NULL |                | `to_book` 内の開始詩番号。 |
| `to_verse_end` |整数 |はい |             | NULL |                | `to_book` の終了詩番号。 |
| `投票` |整数 |はい |             | NULL |                |相互参照の関連性を示す投票数。 |
