# 성경 버전 및 상호 참조 데이터베이스: MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

이것은 다양한 형식의 성경 버전 모음입니다. 다음은 몇 가지 간단한 소개입니다.

- [Formats Folder](.././formats)에서 찾고 있는 성경 버전을 찾으세요.
- [Sources Folder](.././sources)에서 소스 텍스트/리소스를 찾으세요.
- [Scripts Folder](.././scripts)에서 이 프로젝트의 제작/변환 스크립트를 활용하세요.
- https://www.openbible.info/labs/cross-references/에 크고 정확한 상호 참조 자료가 포함되어 있습니다.

간단한 [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md)를 참조하세요.

> **⚠️ 중요: 이 프로젝트의 레거시 버전은 [2024](https://github.com/scrollmapper/bible_databases/tree/2024) 분기에서 사용할 수 있습니다. 2025 브랜치 및 후속 버전에서는 데이터베이스 스키마에 대한 중요한 변경 사항이 구현되었습니다.**

## 데이터베이스 스키마

다음 섹션에서는 이 프로젝트에 사용된 일반 데이터베이스 스키마를 설명합니다.

#### 테이블: `<translation>_books`
이 표에는 해당 성경 번역본의 모든 책이 나열되어 있습니다.

| 열 이름 | 유형 | 널 입력 가능 | 열쇠 | 기본값 | 추가 | 설명 |
|-------------|---------------|----------|-------------|---------|---|---------|
| `id` | 정수 | 아니요 | 기본 키 | NULL | 자동 증가 | 각 도서의 고유 식별자입니다.  |
| `이름` | varchar(255) | 예 |             | NULL |                | 책의 이름입니다.             |

#### 테이블: `<translation>_verses`
이 표에는 해당 성경 번역본의 모든 구절이 포함되어 있습니다.

| 열 이름 | 유형 | 널 입력 가능 | 열쇠 | 기본값 | 추가 | 설명 |
|-------------|---------------|----------|-------------|---------|---|---------|
| `id` | 정수 | 아니요 | 기본 키 | NULL | 자동 증가 | 각 구절의 고유 식별자입니다. |
| `책_ID` | 정수 | 예 | 색인 | NULL |                | 책의 ID입니다(`<translation>_books`의 외래 키). |
| `장` | 정수 | 예 |             | NULL |                | 장 번호입니다.               |
| '운문' | 정수 | 예 |             | NULL |                | 구절 번호입니다.                 |
| `텍스트` | 텍스트 | 예 |             | NULL |                | 구절의 텍스트입니다.            |

#### 테이블: `번역`
이 표에는 사용 가능한 성경 번역본에 대한 정보가 포함되어 있습니다.

| 열 이름 | 유형 | 널 입력 가능 | 열쇠 | 기본값 | 추가 | 설명 |
|---------------|---------------|----------|-------------|---------|-------|------|
| `번역`| varchar(255) | 아니요 | 기본 키 | NULL |       | 번역의 약어입니다. |
| `제목` | varchar(255) | 예 |             | NULL |       | 번역의 전체 제목입니다. |
| `라이센스` | 텍스트 | 예 |             | NULL |       | 번역에 대한 라이센스 정보입니다. |

#### 테이블: `교차_참조`
이 표에는 서로 다른 구절 사이의 상호 참조 데이터가 포함되어 있습니다.

| 열 이름 | 유형 | 널 입력 가능 | 열쇠 | 기본값 | 추가 | 설명 |
|----|---------------|----------|-------------|---------|---|--------------------------|
| `id` | 정수 | 아니요 | 기본 키 | NULL | 자동 증가 | 각 상호 참조 항목의 고유 식별자입니다. |
| `from_book` | varchar(255) | 예 | 색인 | NULL |                | 상호 참조가 시작되는 책입니다. |
| `from_chapter` | 정수 | 예 |             | NULL |                | 'from_book'의 장 번호입니다. |
| 'from_verse' | 정수 | 예 |             | NULL |                | 'from_book'의 절 번호입니다. |
| `to_book` | varchar(255) | 예 | 색인 | NULL |                | 상호 참조가 가리키는 책. |
| `to_chapter` | 정수 | 예 |             | NULL |                | `to_book`의 장 번호입니다. |
| `to_verse_start`| 정수 | 예 |             | NULL |                | `to_book`의 시작 절 번호입니다. |
| `to_verse_end` | 정수 | 예 |             | NULL |                | `to_book`의 끝 절 번호입니다. |
| '투표' | 정수 | 예 |             | NULL |                | 상호 참조의 관련성을 나타내는 투표 수입니다. |
