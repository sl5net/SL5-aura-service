# Versões da Bíblia e bancos de dados de referência cruzada: MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

Esta é uma coleção de versões da Bíblia em diferentes formatos. Aqui estão algumas introduções rápidas:

- Encontre as versões da Bíblia que você procura no [Formats Folder](.././formats).
- Encontre os textos/recursos de origem no [Sources Folder](.././sources).
- Utilize os scripts de produção/conversão deste projeto no [Scripts Folder](.././scripts).
- Um recurso de referência cruzada grande e preciso está incluído em https://www.openbible.info/labs/cross-references/

Veja o [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md) simples.

> **⚠️ Importante: A versão legada deste projeto está disponível na filial [2024](https://github.com/scrollmapper/bible_databases/tree/2024). Observe que mudanças significativas no esquema do banco de dados foram implementadas na ramificação 2025 e nas versões subsequentes.**

## Esquema de banco de dados

As seções a seguir descrevem o esquema geral do banco de dados usado neste projeto:

#### Tabela: `<tradução>_livros`
Esta tabela lista todos os livros da tradução da Bíblia fornecida.

| Nome da coluna | Tipo | Anulável | Chave | Padrão | Extras | Descrição |
|---------|---------------|----------|-------------|---------|----------------|-----------------------------------|
| `id` | interno | NÃO | Chave Primária | NULO | incremento_automático | Identificador exclusivo para cada livro.  |
| `nome` | varchar(255) | SIM |             | NULO |                | O nome do livro.             |

#### Tabela: `<tradução>_versos`
Esta tabela contém todos os versículos da tradução da Bíblia.

| Nome da coluna | Tipo | Anulável | Chave | Padrão | Extras | Descrição |
|---------|---------------|----------|-------------|---------|----------------|-----------------------------------|
| `id` | interno | NÃO | Chave Primária | NULO | incremento_automático | Identificador exclusivo para cada versículo. |
| `livro_id` | interno | SIM | Índice | NULO |                | O ID do livro (chave estrangeira para `<translation>_books`). |
| `capítulo` | interno | SIM |             | NULO |                | O número do capítulo.               |
| `verso` | interno | SIM |             | NULO |                | O número do versículo.                 |
| `texto` | texto | SIM |             | NULO |                | O texto do versículo.            |

#### Tabela: `traduções`
Esta tabela contém informações sobre as traduções da Bíblia disponíveis.

| Nome da coluna | Tipo | Anulável | Chave | Padrão | Extras | Descrição |
|--------------|---------------|----------|-------------|---------|-----------|-----------------|
| `tradução`| varchar(255) | NÃO | Chave Primária | NULO |       | A abreviatura da tradução. |
| `título` | varchar(255) | SIM |             | NULO |       | O título completo da tradução. |
| `licença` | texto | SIM |             | NULO |       | As informações de licença para a tradução. |

#### Tabela: `referências_cruzadas`
Esta tabela contém dados de referência cruzada entre diferentes versículos.

| Nome da coluna | Tipo | Anulável | Chave | Padrão | Extras | Descrição |
|-----------------|---------------|----------|-------------|---------|----------------|---------------------------------------|
| `id` | interno | NÃO | Chave Primária | NULO | incremento_automático | Identificador exclusivo para cada entrada de referência cruzada. |
| `do_livro` | varchar(255) | SIM | Índice | NULO |                | O livro a partir do qual a referência cruzada começa. |
| `do_capítulo` | interno | SIM |             | NULO |                | O número do capítulo em `from_book`. |
| `from_verse` | interno | SIM |             | NULO |                | O número do versículo em `from_book`. |
| `para_reservar` | varchar(255) | SIM | Índice | NULO |                | O livro para o qual a referência cruzada aponta. |
| `para_capítulo` | interno | SIM |             | NULO |                | O número do capítulo em `to_book`. |
| `to_verse_start`| interno | SIM |             | NULO |                | O número do versículo inicial em `to_book`. |
| `to_verse_end` | interno | SIM |             | NULO |                | O número do versículo final em `to_book`. |
| `votos` | interno | SIM |             | NULO |                | O número de votos que indica a relevância da referência cruzada. |
