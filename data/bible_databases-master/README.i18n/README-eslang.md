# Versiones de la Biblia y bases de datos de referencias cruzadas: MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

Esta es una colección de versiones de la Biblia en diferentes formatos. Aquí hay algunas presentaciones rápidas:

- Encuentra las versiones de la biblia que buscas en el [Formats Folder](.././formats).
- Encuentre los textos/recursos fuente en [Sources Folder](.././sources).
- Utilizar los scripts de producción/conversión para este proyecto en [Scripts Folder](.././scripts).
- Se incluye un recurso de referencia cruzada amplio y preciso en https://www.openbible.info/labs/cross-references/

Vea el [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md) simple.

> **⚠️ Importante: La versión heredada de este proyecto está disponible en la rama [2024](https://github.com/scrollmapper/bible_databases/tree/2024). Tenga en cuenta que se han implementado cambios significativos en el esquema de la base de datos en la rama 2025 y versiones posteriores.**

## Esquema de base de datos

Las siguientes secciones describen el esquema de base de datos general utilizado en este proyecto:

#### Tabla: `<traducción>_libros`
Esta tabla enumera todos los libros en la traducción dada de la Biblia.

| Nombre de columna | Tipo | Anulable | Clave | Predeterminado | Adicional | Descripción |
|-------------|---------------|----------|-------------|---------|----------------|-----------------------------------|
| `identificación` | entero | NO | Clave principal | NULO | incremento_automático | Identificador único para cada libro.  |
| `nombre` | varchar(255) | SÍ |             | NULO |                | El nombre del libro.             |

#### Tabla: `<traducción>_versos`
Esta tabla contiene todos los versículos de la traducción dada de la Biblia.

| Nombre de columna | Tipo | Anulable | Clave | Predeterminado | Adicional | Descripción |
|-------------|---------------|----------|-------------|---------|----------------|-----------------------------------|
| `identificación` | entero | NO | Clave principal | NULO | incremento_automático | Identificador único para cada verso. |
| `libro_id` | entero | SÍ | Índice | NULO |                | El ID del libro (clave externa para `<traducción>_books`). |
| `capítulo` | entero | SÍ |             | NULO |                | El número del capítulo.               |
| `verso` | entero | SÍ |             | NULO |                | El número del verso.                 |
| `texto` | texto | SÍ |             | NULO |                | El texto del verso.            |

#### Tabla: `traducciones`
Esta tabla contiene información sobre las traducciones de la Biblia disponibles.

| Nombre de columna | Tipo | Anulable | Clave | Predeterminado | Adicional | Descripción |
|--------------|---------------|----------|-------------|---------|-------|--------------------------------|
| `traducción`| varchar(255) | NO | Clave principal | NULO |       | La abreviatura de la traducción. |
| `título` | varchar(255) | SÍ |             | NULO |       | El título completo de la traducción. |
| `licencia` | texto | SÍ |             | NULO |       | La información de licencia para la traducción. |

#### Tabla: `referencias_cruzadas`
Esta tabla contiene datos de referencias cruzadas entre diferentes versículos.

| Nombre de columna | Tipo | Anulable | Clave | Predeterminado | Adicional | Descripción |
|-----------------|---------------|----------|-------------|---------|----------------|---------------------------------|
| `identificación` | entero | NO | Clave principal | NULO | incremento_automático | Identificador único para cada entrada de referencia cruzada. |
| `desde_libro` | varchar(255) | SÍ | Índice | NULO |                | El libro del que parte la referencia cruzada. |
| `del_capítulo` | entero | SÍ |             | NULO |                | El número de capítulo en el `from_book`. |
| `desde_verso` | entero | SÍ |             | NULO |                | El número de verso en el `from_book`. |
| `para_reservar` | varchar(255) | SÍ | Índice | NULO |                | El libro al que apunta la referencia cruzada. |
| `al_capítulo` | entero | SÍ |             | NULO |                | El número de capítulo en el `to_book`. |
| `to_verse_start`| entero | SÍ |             | NULO |                | El número del verso inicial en el `to_book`. |
| `to_verse_end` | entero | SÍ |             | NULO |                | El número del verso final en el `to_book`. |
| `votos` | entero | SÍ |             | NULO |                | El número de votos que indica la relevancia de la referencia cruzada. |
