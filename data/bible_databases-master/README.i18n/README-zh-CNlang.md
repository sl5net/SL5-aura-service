# 圣经版本和交叉引用数据库：MySQL、SQLite、CSV、JSON、YAML、TXT、MD。

这是不同格式的圣经版本的集合。以下是一些快速介绍：

- 在 [Formats Folder](.././formats) 中找到您正在寻找的圣经版本。
- 在 [Sources Folder](.././sources) 中查找源文本/资源。
- 在 [Scripts Folder](.././scripts) 中利用该项目的生产/转换脚本。
- https://www.openbible.info/labs/cross-references/ 包含大量准确的交叉引用资源

请参阅简单的 [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md)。

> **⚠️ 重要提示：该项目的旧版本可在 [2024](https://github.com/scrollmapper/bible_databases/tree/2024) 分支上找到。请注意，2025 分支及后续版本已对数据库架构进行了重大更改。**

＃＃ 数据库架构

以下部分描述了该项目中使用的一般数据库模式：

#### 表：`<翻译>_books`
该表列出了给定圣经译本中的所有书籍。

|栏目名称 |类型 |可空 |关键|默认 |额外 |描述 |
|------------|--------------|----------|--------------|---------|----------------|------------------------------------|
| `id` |整数 |否 |主键 |空 |自动增量 |每本书的唯一标识符。  |
| `名称` | varchar(255) | varchar(255) |是 |             |空 |                |书名。             |

#### 表：`<翻译>_verses`
该表包含给定圣经译本中的所有经文。

|栏目名称 |类型 |可空 |关键|默认 |额外 |描述 |
|------------|--------------|----------|--------------|---------|----------------|------------------------------------|
| `id` |整数 |否 |主键 |空 |自动增量 |每节经文的唯一标识符。 |
| `book_id` |整数 |是 |索引 |空 |                |书籍的 ID（“<translation>_books”的外键）。 |
| `章节` |整数 |是 |             |空 |                |章节编号。               |
| `诗句` |整数 |是 |             |空 |                |诗句编号。                 |
| `文本` |文字|是 |             |空 |                |诗句的正文。            |

#### 表：`翻译`
此表包含有关可用圣经翻译的信息。

|栏目名称 |类型 |可空 |关键|默认 |额外 |描述 |
|--------------|---------------|----------|----------|----------|--------------------|--------------------------------|
| `翻译`| varchar(255) | varchar(255) |否 |主键 |空 |       |译文的缩写。 |
| `标题` | varchar(255) | varchar(255) |是 |             |空 |       |翻译的完整标题。 |
| `许可证` |文字|是 |             |空 |       |翻译的许可证信息。 |

#### 表：`cross_references`
该表包含不同经文之间的交叉引用数据。

|栏目名称 |类型 |可空 |关键|默认 |额外 |描述 |
|----------------|--------------|----------|-------------|---------|----------------|----------------------------------------|
| `id` |整数 |否 |主键 |空 |自动增量 |每个交叉引用条目的唯一标识符。 |
| `来自书本` | varchar(255) | varchar(255) |是 |索引 |空 |                |交叉引用从这本书开始。 |
| `from_chapter` |整数 |是 |             |空 |                | `from_book` 中的章节编号。 |
| `from_verse` |整数 |是 |             |空 |                | `from_book` 中的诗节编号。 |
| `to_book` | varchar(255) | varchar(255) |是 |索引 |空 |                |交叉引用指向的书。 |
| `到_章节` |整数 |是 |             |空 |                | `to_book` 中的章节编号。 |
| `to_verse_start`|整数 |是 |             |空 |                | `to_book` 中的起始诗节编号。 |
| `到_verse_end` |整数 |是 |             |空 |                | `to_book` 中的结束诗句编号。 |
| `投票` |整数 |是 |             |空 |                |表示交叉引用相关性的投票数。 |
