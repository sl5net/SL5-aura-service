# Wersje Biblii i bazy danych odsyłaczy: MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

Jest to zbiór wersji Biblii w różnych formatach. Oto kilka krótkich prezentacji:

- Znajdź wersje Biblii, których szukasz w [Formats Folder](.././formats).
- Znajdź teksty źródłowe/zasoby w [Sources Folder](.././sources).
- Wykorzystaj skrypty produkcyjne/konwersyjne dla tego projektu w [Scripts Folder](.././scripts).
- Obszerny i dokładny zasób odsyłaczy znajduje się na stronie https://www.openbible.info/labs/cross-references/

Zobacz prosty [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md).

> **⚠️ Ważne: starsza wersja tego projektu jest dostępna w gałęzi [2024](https://github.com/scrollmapper/bible_databases/tree/2024). Informujemy, że w gałęzi 2025 i kolejnych wersjach zaimplementowano istotne zmiany w schemacie bazy danych.**

## Schemat bazy danych

W poniższych sekcjach opisano ogólny schemat bazy danych zastosowany w tym projekcie:

#### Tabela: `<tłumaczenie>_książki`
Ta tabela zawiera listę wszystkich ksiąg danego tłumaczenia Biblii.

| Nazwa kolumny | Wpisz | Możliwość wartości null | Klucz | Domyślne | Ekstra | Opis |
|------------|---------------|----------|------------|---------|----------------|-----------------------------------|
| `id` | int | NIE | Klucz podstawowy | NULL | automatyczny_przyrost | Unikalny identyfikator każdej książki.  |
| `imię` | varchar(255) | TAK |             | NULL |                | Nazwa książki.             |

#### Tabela: `<translation>_verses`
Tabela ta zawiera wszystkie wersety z danego tłumaczenia Biblii.

| Nazwa kolumny | Wpisz | Możliwość wartości null | Klucz | Domyślne | Ekstra | Opis |
|------------|---------------|----------|------------|---------|----------------|-----------------------------------|
| `id` | int | NIE | Klucz podstawowy | NULL | automatyczny_przyrost | Unikalny identyfikator każdego wersetu. |
| `id_książki` | int | TAK | Indeks | NULL |                | Identyfikator książki (klucz obcy do `<translation>_books`). |
| `rozdział` | int | TAK |             | NULL |                | Numer rozdziału.               |
| `wiersz` | int | TAK |             | NULL |                | Numer wersetu.                 |
| `tekst` | tekst | TAK |             | NULL |                | Tekst wersetu.            |

#### Tabela: `tłumaczenia`
Ta tabela zawiera informacje o dostępnych tłumaczeniach Biblii.

| Nazwa kolumny | Wpisz | Możliwość wartości null | Klucz | Domyślne | Ekstra | Opis |
|-------------|---------------|----------|-------------|---------|-------|--------------------------------|
| `tłumaczenie`| varchar(255) | NIE | Klucz podstawowy | NULL |       | Skrót tłumaczenia. |
| `tytuł` | varchar(255) | TAK |             | NULL |       | Pełny tytuł tłumaczenia. |
| `licencja` | tekst | TAK |             | NULL |       | Informacje licencyjne dotyczące tłumaczenia. |

#### Tabela: `odsyłacze`
Ta tabela zawiera dane odsyłaczy między różnymi wersetami.

| Nazwa kolumny | Wpisz | Możliwość wartości null | Klucz | Domyślne | Ekstra | Opis |
|----------------|--------------|----------|-------------|---------|----------------|---------------------------------------|
| `id` | int | NIE | Klucz podstawowy | NULL | automatyczny_przyrost | Niepowtarzalny identyfikator każdego wpisu odsyłacza. |
| `z_książki` | varchar(255) | TAK | Indeks | NULL |                | Książka, od której rozpoczyna się odsyłacz. |
| `z_rozdziału` | int | TAK |             | NULL |                | Numer rozdziału w `from_book`. |
| `z_wersum` | int | TAK |             | NULL |                | Numer wersetu w `from_book`. |
| `do_zarezerwowania` | varchar(255) | TAK | Indeks | NULL |                | Książka, do której odsyła odsyłacz. |
| `do_rozdziału` | int | TAK |             | NULL |                | Numer rozdziału w `to_book`. |
| `do_wersu_start`| int | TAK |             | NULL |                | Numer początkowego wersetu w `to_book`. |
| `do_końca_wersu` | int | TAK |             | NULL |                | Numer końcowego wersetu w `to_book`. |
| `głosy` | int | TAK |             | NULL |                | Liczba głosów wskazująca na istotność powiązania. |
