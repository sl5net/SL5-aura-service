# Versions bibliques et bases de données de références croisées : MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

Il s'agit d'une collection de versions de la Bible dans différents formats. Voici quelques introductions rapides :

- Trouvez les versions bibliques que vous recherchez dans le [Formats Folder](.././formats).
- Retrouvez les textes sources/ressources dans le [Sources Folder](.././sources).
- Utiliser les scripts de production/conversion pour ce projet dans le [Scripts Folder](.././scripts).
- Une ressource de références croisées vaste et précise est incluse à partir de https://www.openbible.info/labs/cross-references/

Voir le simple [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md).

> **⚠️ Important : La version Legacy de ce projet est disponible sur la branche [2024](https://github.com/scrollmapper/bible_databases/tree/2024). Veuillez noter que des modifications importantes ont été apportées au schéma de base de données dans la branche 2025 et les versions ultérieures.**

## Schéma de base de données

Les sections suivantes décrivent le schéma général de base de données utilisé dans ce projet :

#### Tableau : `<translation>_books`
Ce tableau répertorie tous les livres dans la traduction donnée de la Bible.

| Nom de la colonne | Tapez | Nullable | Clé | Par défaut | Supplémentaire | Descriptif |
|-------------|---------------|--------------|-------------|-------------|----------------|-----------------------------------|
| `identifiant` | entier | NON | Clé primaire | NULL | auto_increment | Identifiant unique pour chaque livre.  |
| `nom` | varchar(255) | OUI |             | NULL |                | Le nom du livre.             |

#### Tableau : `<translation>_verses`
Ce tableau contient tous les versets de la traduction donnée de la Bible.

| Nom de la colonne | Tapez | Nullable | Clé | Par défaut | Supplémentaire | Descriptif |
|-------------|---------------|--------------|-------------|-------------|----------------|-----------------------------------|
| `identifiant` | entier | NON | Clé primaire | NULL | auto_increment | Identifiant unique pour chaque verset. |
| `book_id` | entier | OUI | Indice | NULL |                | L'ID du livre (clé étrangère de `<translation>_books`). |
| `chapitre` | entier | OUI |             | NULL |                | Le numéro du chapitre.               |
| `verset` | entier | OUI |             | NULL |                | Le numéro du verset.                 |
| `texte` | texte | OUI |             | NULL |                | Le texte du verset.            |

#### Tableau : `traductions`
Ce tableau contient des informations sur les traductions de la Bible disponibles.

| Nom de la colonne | Tapez | Nullable | Clé | Par défaut | Supplémentaire | Descriptif |
|--------------|---------------|--------------|-------------|-------------|-------|--------------------------------|
| `traduction`| varchar(255) | NON | Clé primaire | NULL |       | L'abréviation de la traduction. |
| `titre` | varchar(255) | OUI |             | NULL |       | Le titre complet de la traduction. |
| `licence` | texte | OUI |             | NULL |       | Les informations de licence pour la traduction. |

#### Tableau : `références_croisées`
Ce tableau contient des données de références croisées entre différents versets.

| Nom de la colonne | Tapez | Nullable | Clé | Par défaut | Supplémentaire | Descriptif |
|-----------------|---------------|--------------|-------------|-------------|----------------|---------------------------------------|
| `identifiant` | entier | NON | Clé primaire | NULL | auto_increment | Identifiant unique pour chaque entrée de référence croisée. |
| `from_book` | varchar(255) | OUI | Indice | NULL |                | Le livre à partir duquel commence la référence croisée. |
| `from_chapter` | entier | OUI |             | NULL |                | Le numéro de chapitre dans le `from_book`. |
| `from_verse` | entier | OUI |             | NULL |                | Le numéro du verset dans le « from_book ». |
| `to_book` | varchar(255) | OUI | Indice | NULL |                | Le livre vers lequel pointe la référence croisée. |
| `vers_chapitre` | entier | OUI |             | NULL |                | Le numéro de chapitre dans `to_book`. |
| `to_verse_start`| entier | OUI |             | NULL |                | Le numéro du verset de départ dans le « to_book ». |
| `to_verse_end` | entier | OUI |             | NULL |                | Le numéro du verset final dans `to_book`. |
| `votes` | entier | OUI |             | NULL |                | Le nombre de votes indiquant la pertinence du renvoi. |
