# Bibelversionen und Querverweisdatenbanken: MySQL, SQLite, CSV, JSON, YAML, TXT, MD.

Dies ist eine Sammlung von Bibelversionen in verschiedenen Formaten. Hier sind einige kurze Einführungen:

- Finden Sie die Bibelversionen, die Sie suchen, im [Formats Folder](.././formats).
- Finden Sie die Quelltexte/Ressourcen im [Sources Folder](.././sources).
- Nutzen Sie die Produktions-/Konvertierungsskripte für dieses Projekt im [Scripts Folder](.././scripts).
- Eine umfangreiche und genaue Querverweisquelle ist unter https://www.openbible.info/labs/cross-references/ enthalten.

Siehe das einfache [DOCUMENTATION](https://github.com/scrollmapper/bible_databases/blob/2025/docs/README.md).

> **⚠️ Wichtig: Die Legacy-Version dieses Projekts ist im [2024](https://github.com/scrollmapper/bible_databases/tree/2024)-Zweig verfügbar. Bitte beachten Sie, dass im Zweig 2025 und nachfolgenden Versionen wesentliche Änderungen am Datenbankschema implementiert wurden.**

## Datenbankschema

In den folgenden Abschnitten wird das allgemeine Datenbankschema beschrieben, das in diesem Projekt verwendet wird:

#### Tabelle: `<translation>_books`
Diese Tabelle listet alle Bücher in der angegebenen Bibelübersetzung auf.

| Spaltenname | Geben Sie | ein Nullable | Schlüssel | Standard | Extra | Beschreibung |
|-------------|---------------|----------|-------------|---------|----------------|---------------------|
| `id` | int | NEIN | Primärschlüssel | NULL | auto_increment | Eindeutige Kennung für jedes Buch.  |
| `Name` | varchar(255) | JA |             | NULL |                | Der Name des Buches.             |

#### Tabelle: `<translation>_verses`
Diese Tabelle enthält alle Verse der angegebenen Bibelübersetzung.

| Spaltenname | Geben Sie | ein Nullable | Schlüssel | Standard | Extra | Beschreibung |
|-------------|---------------|----------|-------------|---------|----------------|---------------------|
| `id` | int | NEIN | Primärschlüssel | NULL | auto_increment | Eindeutige Kennung für jeden Vers. |
| `book_id` | int | JA | Index | NULL |                | Die ID des Buchs (Fremdschlüssel für „<translation>_books“). |
| `Kapitel` | int | JA |             | NULL |                | Die Kapitelnummer.               |
| „Vers“ | int | JA |             | NULL |                | Die Versnummer.                 |
| `Text` | Text | JA |             | NULL |                | Der Text des Verses.            |

#### Tabelle: „Übersetzungen“.
Diese Tabelle enthält Informationen zu den verfügbaren Bibelübersetzungen.

| Spaltenname | Geben Sie | ein Nullable | Schlüssel | Standard | Extra | Beschreibung |
|--------------|---------------|----------|-------------|---------|-------|--------------------------------|
| `Übersetzung`| varchar(255) | NEIN | Primärschlüssel | NULL |       | Die Abkürzung der Übersetzung. |
| `Titel` | varchar(255) | JA |             | NULL |       | Der vollständige Titel der Übersetzung. |
| „Lizenz“ | Text | JA |             | NULL |       | Die Lizenzinformationen für die Übersetzung. |

#### Tabelle: „cross_references“.
Diese Tabelle enthält Querverweisdaten zwischen verschiedenen Versen.

| Spaltenname | Geben Sie | ein Nullable | Schlüssel | Standard | Extra | Beschreibung |
|-----------------|---------------|----------|-------------|---------|----------------|--------------------------------------|
| `id` | int | NEIN | Primärschlüssel | NULL | auto_increment | Eindeutige Kennung für jeden Querverweiseintrag. |
| `from_book` | varchar(255) | JA | Index | NULL |                | Das Buch, von dem der Querverweis ausgeht. |
| `from_chapter` | int | JA |             | NULL |                | Die Kapitelnummer im „from_book“. |
| `from_verse` | int | JA |             | NULL |                | Die Versnummer im „from_book“. |
| `to_book` | varchar(255) | JA | Index | NULL |                | Das Buch, auf das der Querverweis verweist. |
| `to_chapter` | int | JA |             | NULL |                | Die Kapitelnummer im „to_book“. |
| `to_verse_start`| int | JA |             | NULL |                | Die Startversnummer im „to_book“. |
| `to_verse_end` | int | JA |             | NULL |                | Die Endversnummer im „to_book“. |
| „Stimmen“ | int | JA |             | NULL |                | Die Anzahl der Stimmen gibt Aufschluss über die Relevanz des Querverweises. |
