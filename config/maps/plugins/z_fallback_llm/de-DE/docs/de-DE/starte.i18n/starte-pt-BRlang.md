#### Richtig (novo Eingabe):

Você deve inserir todas as letras Schrägstriche (`/`) durante o ponto (`.`) e o final `.py`:

```bash
# Stellen Sie sicher, dass Sie im Projekt-Root-Verzeichnis ~/pr/py/STT sind
python -m config.maps.plugins.z_fallback_llm.de-DE.simulate_conversation
```

### Die Erklärung

* **`python -m`** contém: "O seguinte elemento como **Modul** ou **Package** também."
* Python-Module e -Packages foram adicionados com **Punkt-Notation** (`package.subpackage.module`) endereçado, bem como o ponto da hierarquia.
* Seu módulo é **`simulate_conversation`** e é encontrado no Package-Pfad **`config.maps.plugins.z_fallback_llm.de-DE`**.

Se você usar o recurso correto, deverá usar o recurso de configuração padrão (`Nenhum módulo chamado 'config.maps'`) e o Python usará a verificação de raiz de seus projetos corretamente no formulário atualizado.
