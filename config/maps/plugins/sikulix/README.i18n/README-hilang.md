# कंप्यूटर विज़न द्वारा संचालित विज़ुअल ऑटोमेशन फ्रेमवर्क
**यदि आप इसे देख सकते हैं, तो आप इसे स्वचालित कर सकते हैं।**

सिकुलीएक्स को अब इसके मूल निर्माता [**RaiMan**](https://github.com/RaiMan) की पूर्ण सहमति के साथ, [**oculix-org**](https://github.com/oculix-org) के तहत सक्रिय रूप से बनाए रखा गया है।

## सिकुलीएक्स क्या है
सिकुलीएक्स आपकी स्क्रीन पर दिखाई देने वाली किसी भी चीज़ को पहचानने और उससे इंटरैक्ट करने के लिए **कंप्यूटर विज़न** ([OpenCV](https://opencv.org/) द्वारा संचालित) का उपयोग करता है - **Windows**, **macOS** और **Linux** पर।

यह **छवि पहचान** के माध्यम से जीयूआई तत्वों का पता लगाता है, फिर उन्हें सिम्युलेटेड माउस और कीबोर्ड क्रियाओं के साथ चलाता है। स्रोत कोड, DOM या आंतरिक API तक पहुंच की आवश्यकता नहीं है।

## ऑरा प्लगइन
यह प्लगइन आपको सिकुलीएक्स आईडीई फोकस में होने पर आवाज द्वारा सिकुलीएक्स कमांड को निर्देशित करने की सुविधा देता है।

| वॉइस कमांड (डी) | आउटपुट |
|---|---|
| "क्लिक करें" | `क्लिक("image.png")` |
| "डोपेलक्लिक" | `डबलक्लिक("image.png")` |
| "रेचट्स्क्लिक" | `राइटक्लिक("image.png")` |
| "वार्टे" | `रुको("image.png", 10)` |

कमांड केवल तभी सक्रिय होते हैं जब सिकुलीएक्स विंडो (`सिकुलिक्साइड`, `सिकुलीएक्स`, `सिकुली`) पर ध्यान केंद्रित किया जाता है।
