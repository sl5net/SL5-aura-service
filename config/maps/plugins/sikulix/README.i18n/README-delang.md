# Visual Automation Framework, unterstützt durch Computer Vision
**Wenn Sie es sehen können, können Sie es automatisieren.**

SikuliX wird jetzt aktiv unter [**oculix-org**](https://github.com/oculix-org) gepflegt, mit der vollständigen Zustimmung seines ursprünglichen Erstellers [**RaiMan**](https://github.com/RaiMan).

## Was ist SikuliX
SikuliX nutzt **Computer Vision** (unterstützt von [OpenCV](https://opencv.org/)), um alles, was auf Ihrem Bildschirm sichtbar ist, zu identifizieren und mit ihm zu interagieren – unter **Windows**, **macOS** und **Linux**.

Es findet GUI-Elemente durch **Bilderkennung** und steuert sie dann mit simulierten Maus- und Tastaturaktionen. Kein Zugriff auf Quellcode, DOM oder interne APIs erforderlich.

## Aura-Plugin
Mit diesem Plugin können Sie SikuliX-Befehle per Stimme diktieren, während die SikuliX-IDE im Fokus ist.

| Sprachbefehl (de) | Ausgabe |
|---|---|
| "klicken" | `click("image.png")` |
| "doppelklick" | `doubleClick("image.png")` |
| "rechtsklick" | `rightClick("image.png")` |
| "warte" | `wait("image.png", 10)` |

Befehle sind nur aktiv, wenn ein SikuliX-Fenster („sikulixide“, „SikuliX“, „Sikuli“) fokussiert ist.
