# إطار الأتمتة المرئية المدعوم من رؤية الكمبيوتر
**إذا كان بإمكانك رؤيته، فيمكنك تشغيله تلقائيًا.**

تتم الآن صيانة SikuliX بشكل نشط تحت [**oculix-org**](https://github.com/oculix-org)، بموافقة كاملة من منشئه الأصلي [**RaiMan**](https://github.com/RaiMan).

                                                ## ما هو سيكوليكس
يستخدم SikuliX **رؤية الكمبيوتر** (المدعومة بواسطة [OpenCV](https://opencv.org/)) لتحديد أي شيء مرئي على شاشتك والتفاعل معه — عبر **Windows** و**macOS** و**Linux**.

فهو يحدد موقع عناصر واجهة المستخدم الرسومية من خلال **التعرف على الصور**، ثم يدفعها باستخدام إجراءات محاكاة الماوس ولوحة المفاتيح. لا يلزم الوصول إلى كود المصدر أو DOM أو واجهات برمجة التطبيقات الداخلية.

                                  ## البرنامج المساعد هالة
يتيح لك هذا البرنامج المساعد إملاء أوامر SikuliX عن طريق الصوت أثناء التركيز على SikuliX IDE.

                          | الأمر الصوتي (دي) | الإخراج |
                                                                    |---|---|
                                     | "انقر" | `انقر("image.png")` |
                          | "دوبلكليك" | `doubleClick("image.png")` |
| "rechtsklick" | `النقر بزر الماوس الأيمن("image.png")` |
                               | "ورت" | `انتظر ("image.png"، 10)` |

تكون الأوامر نشطة فقط عند التركيز على نافذة SikuliX (`sikulixide`، `SikuliX`، `Sikuli`).
