Konfigurations-Koans

### 1. Koan 08: Die Schalter-Zentrale (Plugins aktivieren)
**Lerneffekt:** Teilnehmer lernen, wie sie in der `settings.py` neue Funktionen (Plugins) freischalten.

*   **Aufgabe:** "Aktiviere das Wikipedia-Plugin, indem du den Wert von `False` auf `True` änderst."
*   **Nutzen:** Verstehen, dass Aura modular ist.

### 2. Koan 09: Dein digitales Namensschild (Variablen)
**Lerneffekt:** Eigene Daten in der Konfiguration hinterlegen, die von Plugins genutzt werden.
*   **Aufgabe:** "Trage deinen eigenen Namen in die Variable `USER_NAME` ein."
*   **Nutzen:** Plugins können dann Sätze schreiben wie "Mit freundlichen Grüßen, [Dein Name]".

### 3. Koan 10: Geduld bitte! (Pausen-Zeiten)
**Lerneffekt:** Die Spracherkennung an das eigene Sprechtempo anpassen.
*   **Aufgabe:** "Erhöhe den `SPEECH_PAUSE_TIMEOUT`, damit Aura länger wartet, bevor sie deinen Satz verarbeitet."
*   **Nutzen:** Besonders wenn sie in Ruhe nachdenken.
