Konfiguracje-Koany

### 1. Koan 08: Die Schalter-Zentrale (Aktywuj wtyczki)
**Lerneffekt:** Teilnehmer lernen, wie sie w `settings.py` neue Funktionen (Plugins) freeischalten.

* **Aufgabe:** "Aktiviere das Wikipedia-Plugin, indem du den Wert von `False` auf `True` änderst."
* **Nutzen:** Verstehen, modułowy moduł Aura.

### 2. Koan 09: Dein digitales Namensschild (Zmienna)
**Lerneffekt:** Data własna w wskazówce dotyczącej konfiguracji, die von Plugins genutzt werden.
* **Aufgabe:** "Deinen eigenen Traged Name in die Variable `USER_NAME` ein."
* **Nutzen:** Plugins können dann Sätze schreiben wie "Mit freundlichen Grüßen, [Dein Name]".

### 3. Koan 10: Geduld ugryź! (Pausen-Zeiten)
**Lerneffekt:** Die Spracherkennung an das eigene Sprechtempo anpassen.
* **Aufgabe:** "Erhöhe den `SPEECH_PAUSE_TIMEOUT`, dajit Aura länger wartt, bevor sie deinen Satz verarbeitet."
* **Nutzen:** Besonders wenn sie in Ruhe nachdenken.
