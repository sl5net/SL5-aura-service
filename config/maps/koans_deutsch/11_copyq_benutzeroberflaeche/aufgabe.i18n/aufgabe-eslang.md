### Koan 11 – Panel de control Dein Sprach
**Lerneffekt:** Teilnehmer lernen, wie sie ihre eigenen Regeln visualisieren und durchsuchbar machen.

**Aufgabenstellung:**
1. **Instaladores:** Lade CopyQ se inicia y se ejecuta.
2. **Taggen:** Offne dein Terminal und führe den Tagger aus.
3. **Exportadores:** Schicke deine Regeln an das Dashboard.
4. **Buscar:** Busque en CopyQ (con `Strg+F`) después del Regel "mfg" (en Koan 09).

**Terminal-Befehle für die Teilnehmer:**
```powershell
python tools/map_tagger.py
python tools/export_to_copyq.py
```
