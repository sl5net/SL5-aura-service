4. यूट्यूब-ट्रांसक्रिप्ट वर्जन से डेन आउटपुट

**सिंक डेन डेन वॉर्हेर अर्स्टेल्ट हेबेन।**

इससे पहले कि आप `मोड == 'SYSTEM_DEFAULT'` (कोई `MIC_AND_DESKTOP`) सिंक न करें।

### वर्सुचटे लोसुंगेन

| वर्सुच | समस्या |
|---|---|
| पल्सऑडियो वर्चुअल सोर्स पहले | पाइपवायर 'मॉड्यूल-वर्चुअल-सोर्स' को अनदेखा करता है |
| `settings_local.py` auf `MIC_AND_DESKTOP` setzen | डेटी वुर्डे मिट मेहरफैचेन इंट्राजेन कोर्रम्पिएर्ट |
| मार्किएरटेन ओवरराइड-ब्लॉक और एंडे श्रेइबेन | ऑरा लैड्ट सेटिंग्स निच्ट श्नेल जेनुग न्यू बेवर ट्रिगर कोमट |
| `_create_mic_and_desktop_sink()` सीधे परीक्षण में | `manage_audio_routing` सेशन-स्टार्ट से पहले |
| `पीडब्ल्यू-लूपबैक` | एर्सचेइंट अल्स सोर्स एबर ऑरा होर्ट निच्ट डारौफ |

### Warum `settings_local.py` ओवरराइड कोई कार्य नहीं

`dynamic_settings.py` दिनांक और अंतिम तिथि पर वापस आ गया - एक अंतराल के साथ। डेर ट्रिगर कोमट ज़ू श्नेल नच डेम श्रेइबेन। `SYSTEM_DEFAULT` के साथ सत्र शुरू होने से पहले।

अधिक: जब तक आभा `MIC_AND_DESKTOP` स्थापित नहीं हो जाती, तब तक सिंक पहले ही शुरू हो चुका है **नचस्ट** सत्र-प्रारंभ - कुछ भी नहीं।
