## Gesamt-Tool を使用したミニマリストの操作

### オプション 1: Fokus auf die Architektur (Die Stärke)

> **SL5 Aura サービス: Konfigurierbarer Offline-Sprachassistent。**
>
> * **100% オフライン:** Vosk STT および Lokaler LanguageTool-Server。
> * **Kontrollierte Verarbeitung:** Sequenzielle、kaskadierende Regelausführung (ファジー マップ)。
> * **コマンド:** フルマッチ (`^$`) ストップクリテリウムによる優先順位。
> * **アンパスン:** 構成 (Regel-Tupel) ersetzt Programmierung。
> * **説明:** Einfache Plug-in-Struktur (niedrige Priorität)。

### オプション 2: フォクス・アウフ・デン・ヌッツェン (本当だった)

> **SL5 オーラ: 二等兵とコマンド部隊の死。**
>
> * Sprachbefehle und Diktat – 100% プライベート、オーネ クラウド。
> * Intelligente Korrektur: Grammatik (LT) + Custom-Regelwerk。
> * 柔軟なテキスト変換 (Code-Konventionen、Abkürzungen)。
> * コントロールバー: Einfache `RegEx` - 定義された複雑なロジック。

## ツール全体の最小限の説明

### オプション 1: アーキテクチャと制御に焦点を当てる (テクニカル USP)

> **SL5 Aura サービス: 構成可能なオフライン音声アシスタント。**
>
> * **100% オフライン:** Vosk STT およびローカル言語ツール サーバー。
> * **制御された処理:** 順次、カスケード ルールの実行 (ファジー マップ)。
> * **コマンド:** 完全一致 (`^$`) 停止基準による厳密な優先順位。
> * **カスタマイズ:** 構成 (ルール タプル) は、従来のコーディングを置き換えます。
> * **拡張可能:** シンプルなプラグイン構造 (ファイル順序による優先順位が低い)。

### オプション 2: 機能とプライバシーに重点を置く (ユーザーの利点)

> **SL5 Aura: プライベートディクテーションとコマンドコントロール。**
>
> * 音声コマンドとディクテーション – 100% プライベート、クラウドなし。
> * インテリジェントな修正: 文法 (LT) + カスタム ルール エンジン。
> * 柔軟なテキスト変換 (コード規則、略語)。
> * 完全に制御: 単純な RegEx ルールが複雑なロジックを定義します。
