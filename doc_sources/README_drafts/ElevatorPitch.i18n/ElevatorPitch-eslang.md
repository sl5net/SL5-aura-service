## Minimalistische Erklärungen für das Gesamt-Tool

### Opción 1: Fokus auf die Architektur (Die Stärke)

> **Servicio SL5 Aura: Konfigurierbarer Offline-Sprachassistent.**
>
> * **100% sin conexión:** Vosk STT y Lokaler LanguageTool-Server.
> * **Kontrollierte Verarbeitung:** Sequenzielle, kaskadierende Regelausführung (Mapas difusos).
> * **Comandos:** Strikte Priorität durch Full Match (`^$`) Stopp-Kriterium.
> * **Anpassung:** Konfiguration (Regel-Tupel) ersetzt Programmierung.
> * **Erweiterbar:** Einfache Plug-in-Struktur (niedrige Priorität).

### Opción 2: Fokus auf den Nutzen (Was es tut)

> **SL5 Aura: Die Private Diktier- und Kommandosteuerung.**
>
> * Sprachbefehle und Diktat – 100% privado, sin nube.
> * Intelligente Korrektur: Grammatik (LT) + Custom-Regelwerk.
> * Transformación de texto flexible (Code-Konventionen, Abkürzungen).
> * Barra de control: Einfache `RegEx`-Regeln definieren komplex Logik.

## Explicaciones minimalistas para toda la herramienta

### Opción 1: centrarse en la arquitectura y el control (la PVU técnica)

> **Servicio SL5 Aura: Asistente de voz configurable sin conexión.**
>
> * **100% sin conexión:** Vosk STT y servidor Local LanguageTool.
> * **Procesamiento controlado:** Ejecución de reglas secuencial en cascada (Fuzzy Maps).
> * **Comandos:** Prioridad estricta mediante el criterio de parada de coincidencia completa (`^$`).
> * **Personalización:** La configuración (tuplas de reglas) reemplaza la codificación tradicional.
> * **Extensible:** Estructura de complemento simple (baja prioridad según el orden de los archivos).

### Opción 2: centrarse en la funcionalidad y la privacidad (el beneficio para el usuario)

> **SL5 Aura: Dictado Privado y Control de Comandos.**
>
> * Comandos de voz y dictados: 100% privado, sin nube.
> * Corrección Inteligente: Gramática (LT) + Motor de reglas personalizadas.
> * Transformación de texto flexible (convenciones de código, abreviaturas).
> * Totalmente controlado: las reglas RegEx simples definen una lógica compleja.
