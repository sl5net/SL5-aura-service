## 개발자 가이드: 고급 퍼지 맵 정규식

퍼지 매핑 시스템은 표준 Python 정규 표현식을 활용하여 특히 **부정 예측(`(?!...)`)**을 통해 강력한 일치 및 제외 패턴을 허용합니다.

### 화이트리스트에 부정적 예측 사용

이 패턴을 사용하면 특정 단어 또는 구문 목록을 제외한 모든 항목에 적용되는 규칙을 정의할 수 있습니다. 이는 `empty_all` 패턴과 함께 누적되고 제한된 규칙 세트를 구축하는 데 특히 유용합니다.

| 목표 | 예제 규칙(`FUZZY_MAP`) | 설명 |
| :--- | :--- | :--- |
| **한 단어를 제외한 모든 항목에 적용** | `('', r'^(?!Haus).*$', 5, {'flags': re.IGNORECASE})` | 이 규칙은 정확하게 "Haus"가 *아닌* **모든 텍스트**에 대체 항목(여기서는 `''` 논리 건너뛰기)을 적용합니다. `(?!Haus)`는 Negative Lookahead로, 텍스트가 "Haus"로 시작하지 않도록 합니다. |
| **여러 단어를 제외한 모든 단어에 적용** | `('', r'^(?!Schach|Matt|bad|Haus).*$', 5, {'flags': re.IGNORECASE})` | 이 규칙은 "Schach", "Matt", "bad" 또는 "Haus"로 시작하지 않는 **모든 것**에 적용됩니다. 여러 용어를 허용 목록에 추가하려면 미리보기 그룹 `(?!...)` 내에서 OR 파이프(`|`)를 사용하세요. |

***

### 제한된 규칙에 대해 긍정적 예측 사용

표준 접근 방식에서는 긍정적 예측 또는 단순 캡처 그룹을 사용하여 특정 단어 목록 *만*으로 규칙을 제한합니다.

| 목표 | 예제 규칙(`FUZZY_MAP`) | 설명 |
| :--- | :--- | :--- |
| **특정 목록에만 적용** | `('Schachmatt', r'^(Schach|Matt|bad|Haus).*$', 5, {'flags': re.IGNORECASE})` | 이 규칙은 텍스트가 나열된 단어(Schach, Matt, bad 또는 Haus) 중 하나로 시작하는 경우에만 적용됩니다. 그런 다음 일치하는 텍스트는 임계값에 따라 대상(`Schachmatt`)으로 대체됩니다. |
