## 开发人员指南：高级模糊映射正则表达式

模糊映射系统利用标准 Python 正则表达式，允许强大的匹配和排除模式，特别是通过**负向查找 (`(?!...)`)**。

### 使用否定前瞻进行白名单

此模式允许您定义适用于**除**特定单词或短语列表之外的所有内容的规则。这与“empty_all”模式结合起来特别有用，可以构建累积的、受限的规则集。

|目标|示例规则 (`FUZZY_MAP`) |说明|
| :--- | :--- | :--- |
| **适用于除一个单词之外的所有单词** | `('', r'^(?!Haus).*$', 5, {'flags': re.IGNORECASE})` |此规则将对“不”完全是“Haus”的**任何文本**应用替换（或跳过逻辑，此处为“''”）。 `(?!Haus)` 是负向前瞻，确保文本不以“Haus”开头。 |
| **适用于除多个单词之外的所有单词** | `('', r'^(?!Schach|Matt|bad|Haus).*$', 5, {'flags': re.IGNORECASE})` |此规则适用于不以“Schach”、“Matt”、“bad”或“Haus”开头的**所有内容**。在前瞻组“(?!...)”中使用 OR 管道 (“|”) 将多个术语列入白名单。 |

***

### 对受限规则使用正向前瞻

标准方法使用正向前瞻或简单的捕获组将规则限制为*仅*特定的单词列表。

|目标|示例规则 (`FUZZY_MAP`) |说明|
| :--- | :--- | :--- |
| **仅适用于特定列表** | `('Schachmatt', r'^(Schach|Matt|bad|Haus).*$', 5, {'flags': re.IGNORECASE})` |此规则仅适用于文本以列出的单词之一（Schach、Matt、bad 或 Haus）开头的情况。然后，匹配的文本将根据阈值替换为目标（“Schachmatt”）。 |
