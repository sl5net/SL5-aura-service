## 開発者ガイド: 高度なファジー マップの正規表現

ファジー マッピング システムは標準の Python 正規表現を利用しており、特に **Negative Lookaheads (`(?!...)`)** を通じて、強力な一致および除外パターンを可能にします。

### ホワイトリストへの否定先読みの使用

このパターンを使用すると、**特定の単語または語句のリストを除くすべて**に適用されるルールを定義できます。これは、「empty_all」パターンと組み合わせて、累積的で制限されたルール セットを構築する場合に特に便利です。

|目標 |ルールの例 (`FUZZY_MAP`) |説明 |
| :--- | :--- | :--- |
| **1 つの単語を除くすべてに適用** | `('', r'^(?!Haus).*$', 5, {'flags': re.IGNORECASE})` |このルールは、正確に「Haus」ではない **任意のテキスト** に置換 (またはスキップ ロジック、ここでは「''」) を適用します。 `(?!Haus)` は否定先読みで、テキストが「Haus」で始まらないようにします。 |
| **複数の単語を除くすべてに適用** | `('', r'^(?!Schach|Matt|bad|Haus).*$', 5, {'flags': re.IGNORECASE})` |このルールは、「Schach」、「Matt」、「bad」、「Haus」で始まらない**すべて**に適用されます。複数の用語をホワイトリストに登録するには、先読みグループ `(?!...)` 内で OR パイプ (`|`) を使用します。 |

***

### 制限されたルールに対する肯定的な先読みの使用

標準的なアプローチでは、Positive Lookaheads または単純なキャプチャ グループを使用して、ルールを特定の単語のリスト *のみ* に制限します。

|目標 |ルールの例 (`FUZZY_MAP`) |説明 |
| :--- | :--- | :--- |
| **特定のリストにのみ適用** | `('シャッハマット', r'^(シャッハ|マット|悪い|ハウス).*$', 5, {'flags': re.IGNORECASE})` |このルールは、テキストがリストされている単語 (Schach、Matt、bad、または Haus) のいずれかで始まる場合にのみ適用されます。一致したテキストは、しきい値に基づいてターゲット (`Schachmatt`) に置き換えられます。 |
