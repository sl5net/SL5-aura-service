# دليل إعداد VirtualBox لاختبار مشروع STT

يوفر هذا الدليل الخطوات الموصى بها لإعداد جهاز ظاهري Ubuntu 24.04 مستقر وفعال في VirtualBox. سيؤدي اتباع هذه الإرشادات إلى إنشاء بيئة متسقة لاختبار تطبيق STT وتجنب المشكلات الشائعة مثل بطء التثبيت وتجميد النظام ووظائف الحافظة المفقودة.

                                       ## المتطلبات الأساسية

                      - VirtualBox مثبت على الجهاز المضيف.
                           - تم تنزيل ملف Ubuntu 24.04 Desktop ISO.

                            ## الأجهزة المضيفة المرجعية

تم اختبار هذا التكوين والتحقق من صحته على النظام المضيف التالي. قد يختلف الأداء على الأجهزة الأخرى، ولكن يجب أن يتم تطبيق إعدادات الاستقرار بشكل عام.

                     - **نظام التشغيل:** مانجارو لينكس
                                                   - **النواة:** 6.6.94
                                 - **المعالج:** 16× AMD Ryzen 7 3700X
- **الذاكرة:** 31.3 جيجا بايت من ذاكرة الوصول العشوائي
                - **معالج الرسومات:** NVIDIA GeForce GTX 1050 Ti

## الجزء الأول: إنشاء الجهاز الافتراضي وتكوينه

تعتبر هذه الإعدادات ضرورية للأداء والاستقرار.

    ### الخطوة 1.1: إنشاء الجهاز الظاهري الجديد

                           1. في VirtualBox، انقر فوق **جديد**.
                2. **الاسم:** `Ubuntu STT Tester\' (أو ما شابه).
                3. **صورة ISO:** اترك هذا الحقل فارغًا.
 4. حدد المربع: **"تخطي التثبيت غير المراقب"**.
                                                5. انقر **التالي**.
                                                       6. **الأجهزة:**
- **الذاكرة الأساسية:** `4096 ميجابايت` أو أكثر.
                                 - **المعالجات:** `4` أو أكثر.
                                                7. انقر **التالي**.

### الخطوة 1.2: إنشاء القرص الصلب الافتراضي (حرج)

هذه هي الخطوة الأكثر أهمية للتثبيت والأداء السريع.

          1. حدد **"إنشاء قرص ثابت افتراضي الآن"**.
  2. اضبط حجم القرص على **40 جيجابايت** أو أكثر.
3. في الشاشة التالية، قم بتغيير نوع التخزين إلى **"الحجم الثابت"**.
> **لماذا؟** يتم تخصيص القرص ذي الحجم الثابت مسبقًا ويمنع اختناق الإدخال/الإخراج الهائل الذي يحدث عندما يتم تغيير حجم القرص "المخصص ديناميكيًا" باستمرار أثناء التثبيت.
4. انقر فوق **إنشاء** وانتظر حتى تنتهي العملية.

### الخطوة 1.3: إعدادات الجهاز الافتراضي النهائية

حدد الجهاز الافتراضي الذي تم إنشاؤه حديثًا وانقر فوق **الإعدادات**. قم بتكوين ما يلي:

                                 - **النظام -> اللوحة الأم:**
                                                 - **الشرائح:** `ICH9`
       - حدد **"تمكين EFI (أنظمة تشغيل خاصة فقط)"**.

                                            - **العرض -> الشاشة:**
                     - **وحدة التحكم بالرسومات:** `VMSVGA`
- **قم بإلغاء تحديد "تمكين التسريع ثلاثي الأبعاد"**.
> **لماذا؟** يعد التسريع ثلاثي الأبعاد سببًا شائعًا لتوقف النظام وتجميده لدى ضيوف Linux. يؤدي تعطيله إلى تحسين الاستقرار بشكل ملحوظ.

                                                          -   **تخزين:**
- حدد **وحدة تحكم SATA**. حدد المربع **"استخدام ذاكرة التخزين المؤقت للإدخال/الإخراج للمضيف"**.
- حدد ملف القرص الظاهري (`.vdi`). حدد المربع **"محرك الحالة الصلبة"**.
- حدد محرك الأقراص الضوئية **الفارغ**. انقر على أيقونة القرص المضغوط الموجودة على اليمين ثم **"اختر ملف قرص..."** لإرفاق Ubuntu 24.04 ISO الخاص بك.

                انقر **موافق** لحفظ كافة الإعدادات.

## الجزء الثاني: تثبيت نظام التشغيل أوبونتو

                      1. ابدأ تشغيل الجهاز الافتراضي.
               2. تابع إعداد اللغة ولوحة المفاتيح.
3. عندما تصل إلى "التحديثات والبرامج الأخرى"، حدد:
                             - **الحد الأدنى من التثبيت**.
- **قم بإلغاء تحديد** "تنزيل التحديثات أثناء تثبيت Ubuntu".
               4. تابع عملية التثبيت حتى اكتمالها.
5. عند الانتهاء، أعد تشغيل الجهاز الافتراضي. عند المطالبة، قم بإزالة وسيط التثبيت (اضغط على Enter).

## الجزء 3: مرحلة ما بعد التثبيت (إضافات الضيف)

تتيح هذه الخطوة مشاركة الحافظة والسحب والإفلات وتغيير حجم الشاشة تلقائيًا.

### الخطوة 3.1: تثبيت إضافات الضيف ISO على المضيف (إذا لزم الأمر)

على **الجهاز المضيف** الخاص بك، تأكد من تثبيت حزمة Guest Additions ISO.

                                      - **أون آرتش / مانجارو:**
    ```bash
    sudo pacman -S virtualbox-guest-iso
    ```
                                    - **على ديبيان/أوبونتو:**
    ```bash
    sudo apt install virtualbox-guest-additions-iso
    ```

  ### الخطوة 3.2: تثبيت إضافات الضيف داخل Ubuntu VM

قم بتنفيذ هذه الخطوات ** داخل جهاز Ubuntu VM قيد التشغيل **.

1. **تحضير Ubuntu:** افتح الوحدة الطرفية وقم بتشغيل الأوامر التالية لتثبيت تبعيات البناء.
    ```bash
    sudo apt update
    sudo apt install build-essential dkms linux-headers-$(uname -r)
    ```
2. **أدخل القرص المضغوط:** من القائمة العلوية لـ VirtualBox، انتقل إلى **الأجهزة -> أدخل صورة القرص المضغوط لإضافات الضيف...**.
                        3. **قم بتشغيل برنامج التثبيت:**
- قد يظهر مربع حوار يطلب تشغيل البرنامج. انقر فوق **تشغيل**.
- إذا لم يظهر أي مربع حوار، افتح مدير الملفات، وانقر بزر الماوس الأيمن فوق القرص المضغوط `VBox_GAs...`، واختر **"فتح في الوحدة الطرفية"**، وقم بتشغيل الأمر:
      ```bash
      sudo ./VBoxLinuxAdditions.run
      ```
4. **إعادة التشغيل:** بعد اكتمال التثبيت، أعد تشغيل الجهاز الافتراضي.
    ```bash
    reboot
    ```
5. **تمكين الميزات:** بعد إعادة التشغيل، انتقل إلى قائمة **الأجهزة** وقم بتمكين **الحافظة المشتركة -> ثنائية الاتجاه** و**السحب والإسقاط -> ثنائي الاتجاه**.

بيئة الاختبار المستقرة عالية الأداء جاهزة الآن.
