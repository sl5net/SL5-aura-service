# STT 项目测试的 VirtualBox 设置指南

本指南提供了在 VirtualBox 中设置稳定且高性能的 Ubuntu 24.04 虚拟机的建议步骤。遵循这些说明将为测试 STT 应用程序创建一致的环境，并避免安装缓慢、系统冻结和缺少剪贴板功能等常见问题。

## 先决条件

- VirtualBox 安装在主机上。
- 下载 Ubuntu 24.04 桌面 ISO 文件。

## 参考主机硬件

此配置已在以下主机系统上进行了测试和验证。其他硬件的性能可能会有所不同，但稳定性设置应该普遍适用。

- **操作系统：** Manjaro Linux
- **内核：** 6.6.94
- **处理器：** 16 × AMD Ryzen 7 3700X
- **内存：** 31.3 GiB RAM
- **图形处理器：** NVIDIA GeForce GTX 1050 Ti

---

## 第 1 部分：VM 创建和配置

这些设置对于性能和稳定性至关重要。

### 步骤 1.1：创建新的虚拟机

1. 在 VirtualBox 中，单击“**新建**”。
2. **名称：** `Ubuntu STT Tester`（或类似名称）。
3. **ISO 映像：** 将此字段留空。
4. 选中复选框：**“跳过无人值守安装”**。
5. 单击**下一步**。
6. **硬件：**
- **基本内存：** `4096 MB` 或更多。
- **处理器：** `4` 或更多。
7. 单击“**下一步**”。

### 步骤 1.2：创建虚拟硬盘（关键）

这是快速安装和性能的最重要步骤。

1. 选择**“立即创建虚拟硬盘”**。
2. 将磁盘大小设置为 **40 GB** 或更大。
3. 在下一个屏幕上，将存储类型更改为**“固定大小”**。
> **为什么？** 固定大小的磁盘是预先分配的，可以防止“动态分配”磁盘在安装过程中不断调整大小时出现的大量 I/O 瓶颈。
4. 单击“**创建**”并等待该过程完成。

### 步骤 1.3：最终虚拟机设置

选择新创建的虚拟机，然后单击“**设置**”。配置以下内容：

- **系统 -> 主板：**
- **芯片组：** `ICH9`
- 选中**“启用 EFI（仅限特殊操作系统）”**。

- **显示 -> 屏幕：**
- **图形控制器：** `VMSVGA`
- **取消选中“启用 3D 加速”**。
> **为什么？** 3D 加速是 Linux 客户机中系统挂起和冻结的常见原因。禁用它可以显着提高稳定性。

-   **贮存：**
- 选择 **SATA 控制器**。选中复选框**“使用主机 I/O 缓存”**。
- 选择虚拟磁盘文件 (`.vdi`)。选中复选框**“固态驱动器”**。
- 选择**空**光驱。单击右侧的 CD 图标并 **“选择磁盘文件...”** 以附加 Ubuntu 24.04 ISO。

单击 **确定** 保存所有设置。

---

## 第 2 部分：Ubuntu 操作系统安装

1. 启动虚拟机。
2. 继续进行语言和键盘设置。
3. 当您到达“更新和其他软件”时，选择：
- **最小化安装**。
- **取消选中**“安装 Ubuntu 时下载更新”。
4. 继续安装直至完成。
5. 完成后，重新启动虚拟机。根据提示，删除安装介质（按 Enter 键）。

---
