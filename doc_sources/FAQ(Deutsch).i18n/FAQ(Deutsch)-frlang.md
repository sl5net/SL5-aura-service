#### **Teil 1: Allgemeine Fragen**

**F : Était-ce le SL5 Auro ?**
R : SL5 Auro est un programme de mise à niveau hors ligne de chaque système. R : "Hors ligne" est la méthode de planification appropriée directement au niveau de votre ordinateur. Dies bietet maximale Privatsphäre and Sicherheit, idéal pour les informations vertrauliche (par exemple B. für Anwälte, Ärzte, Journalisten) et est entièrement conforme aux Datenschutzverordnungen wie der DSGVO.

**F : Est-ce que c'est vraiment gratuit ? Où est le Haken ?**
R : La "Community Edition" est 100 % gratuite et open source. C'est un keinen Haken. Nous vous recommandons d'étudier les outils Open-Source. Lorsque vous trouvez le logiciel rapidement et que vous recherchez des solutions plus pratiques, vous pouvez vous assurer de trouver votre [Ko-fi Seite](https://ko-fi.com/sl5).

# Live Hot-Reload pour la configuration

SL5 Aura offre une fonction de recharge à chaud Live pour les configurations, comme par exemple. l'Aktivierung ou la Deaktivierung von Git-Kommandos. Ce service vous permettra de démarrer sans le service SL5 Aura un nouveau moyen – une plus grande option pour la production !

Quelle est la fonction :
Pour obtenir des performances optimales, les configurations doivent être effectuées avant le démarrage d'un nouveau processus de vérification et d'amélioration. Le bedeutet:

1 Änderung speichern : Sie ändern eine Einstellung (par exemple, aktivieren Git-Kommandos).
2 Aktivierung: Die Änderung wird aktiv, sobald SL5 Aura das nächste Mal eine Aktion verarbeitet (par exemple Sie sprechen einen Befehl ein). Le cache sera alors activé.

Remarque importante : Vos produits sont si forts dans les "vorgemerkt" de vente, qu'ils sont d'abord avec l'activité d'interaction la plus proche. C'est le Neustart des Dienstes erforderlich qui vous convient.
