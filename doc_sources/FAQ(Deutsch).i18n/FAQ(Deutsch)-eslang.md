# Live Hot-Reload para configuración

SL5 Aura bietet eine leistungsstarke Live Hot-Reload-Funktion for Konfigurationsänderungen, wie z.B. die Aktivierung oder Deaktivierung von Git-Kommandos. Dies bedeutet, dass Sie Anpassungen vornehmen vornehmen können, ohne den SL5 Aura-Service neu starten zu müssen – ein großer Vorteil für die Produktivität!

Cómo funciona:
Para obtener un rendimiento óptimo, las configuraciones de configuración se realizarán primero al iniciar una nueva configuración de configuración de forma excesiva y mejorada. El bedeutet:

1 Änderung speichern: Sie ändern eine Einstellung (por ejemplo, activar Git-Kommandos).
2 Aktivierung: Die Änderung wird aktiv, sobald SL5 Aura das nächste Male eine Aktion verarbeitet (z.B. Sie sprechen einen Befehl ein). El caché se actualizará.

Wichtiger Hinweis: Ihre Änderungen sind sofort nach dem Speichern "vorgemerkt", werden aber erst mit der nächsten Interaktion aktiv. Es ist kein Neustart des Dienstes erforderlich.
