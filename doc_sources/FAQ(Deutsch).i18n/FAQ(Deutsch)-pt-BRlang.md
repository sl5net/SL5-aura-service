# Live Hot-Reload para configurações

SL5 Aura oferece uma função de atualização ao vivo Hot-Reload para configurações de configuração, como z.B. a ativação ou desativação do comando Git. É bom que você saiba que o SL5 Aura-Service nunca começou a ser necessário – uma oferta maior para a produtividade!

Como funciona:
Para obter um desempenho ideal, as configurações de configuração primeiro serão iniciadas e serão exibidas novas configurações de configuração. O que você precisa:

1 Änderung speichern: Sie ändern eine Einstellung (z.B. ativar comandos Git).
2 Atividade: Die Änderung wird active, sobald SL5 Aura das nächste Mal eine Aktion verarbeitet (z.B. Sie sprechen einen Befehl ein). O cache será atualizado.

Wichtiger Hinweis: Ihre nderungen sind sofort nach dem Speichern "vorgemerkt", werden aber primeiro mit der nächsten Interaktiv. Es ist kein Neustart des Dienstes erforderlich.
