bashsudo pacman -S python-libtorrent
C++-Bibliothek(libtorrent-rasterbar)을 설치하고 패키지 관리자의 공식 Schritt에서 Python-Bindings를 설치하면 됩니다.

sudo pacman -S python-libtorrent

다나흐 테스트:
bashpython3 -c "libtorrent 가져오기; 인쇄(libtorrent.version)"
