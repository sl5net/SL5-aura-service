Pourquoi 0 AD utilise-t-il des torrents ?

Le moyen le plus simple de distribuer un fichier est de le placer sur un serveur central. Ensuite, tous les utilisateurs peuvent le télécharger depuis ce serveur. Mais lorsque de nombreuses personnes téléchargent les fichiers, un seul serveur ne suffit pas et l’installation de différents serveurs à travers le monde s’avère coûteuse. En tant que jeu gratuit, nous ne sommes pas en mesure de payer ces frais.
