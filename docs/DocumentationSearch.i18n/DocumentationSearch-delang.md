# So finden Sie Hilfe
Sie können die Dokumentation mit Ihrer Stimme durchsuchen.
Sagen Sie: „Aura Handbuch“ oder „Aura Dokumentation“.
Es öffnet sich ein Suchfenster.
Es werden nur Hilfedateien in Ihrer Sprache angezeigt.
Verwenden Sie die Pfeiltasten, um ein Thema auszuwählen.
Drücken Sie die Eingabetaste, um das vollständige Dokument zu öffnen.
