# Comment trouver de l'aide
Vous pouvez rechercher la documentation avec votre voix.
Dites : « Aura Handbuch » ou « Aura Dokumentation ».
Une fenêtre de recherche s'ouvrira.
Il affiche uniquement les fichiers d'aide dans votre langue.
Utilisez les touches fléchées pour sélectionner un sujet.
Appuyez sur Entrée pour ouvrir le document complet.
