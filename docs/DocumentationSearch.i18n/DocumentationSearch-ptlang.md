# Como encontrar ajuda
Você pode pesquisar a documentação com sua voz.
Diga: "Aura Handbuch" ou "Aura Documentation".
Uma janela de pesquisa será aberta.
Ele mostra apenas arquivos de ajuda no seu idioma.
Use as teclas de seta para selecionar um tópico.
Pressione Enter para abrir o documento completo.
