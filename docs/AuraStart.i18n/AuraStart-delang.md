# Aura-Startanleitung
Aura startet automatisch mit Ihrem Computer, sofern konfiguriert.
Wenn Aura startet, öffnet sich ein Willkommensfenster.
Dieses Fenster zeigt Ihnen, dass Aura bereit ist.
Drücken Sie Ihren globalen Hotkey, um mit dem Sprechen zu beginnen.
Sprechen Sie deutlich, um Befehle zu senden oder Text einzugeben.
