# Guide de démarrage Aura
Aura démarre automatiquement avec votre ordinateur s'il est configuré.
Lorsque Aura démarre, une fenêtre de bienvenue s'ouvre.
Cette fenêtre vous montre qu'Aura est prête.
Appuyez sur votre touche de raccourci globale pour commencer à parler.
Parlez clairement pour envoyer des commandes ou saisir du texte.
