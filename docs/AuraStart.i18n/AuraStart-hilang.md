# ऑरा स्टार्ट गाइड
यदि कॉन्फ़िगर किया गया है तो ऑरा आपके कंप्यूटर से स्वचालित रूप से प्रारंभ हो जाता है।
जब आभा शुरू होती है, तो एक स्वागत विंडो खुलती है।
यह विंडो आपको दिखाती है कि ऑरा तैयार है।
बोलना शुरू करने के लिए अपनी ग्लोबल हॉटकी दबाएँ।
आदेश भेजने या टेक्स्ट टाइप करने के लिए स्पष्ट रूप से बोलें।
