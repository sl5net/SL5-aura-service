# Guía de inicio de Aura
Aura se inicia automáticamente con su computadora si está configurada.
Cuando se inicia Aura, se abre una ventana de bienvenida.
Esta ventana le muestra que Aura está lista.
Presione su tecla de acceso rápido global para comenzar a hablar.
Habla claramente para enviar comandos o escribir texto.
