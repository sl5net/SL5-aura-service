# Guia inicial da Aura
O Aura inicia automaticamente com o seu computador, se configurado.
Quando o Aura é iniciado, uma janela de boas-vindas é aberta.
Esta janela mostra que o Aura está pronto.
Pressione sua tecla de atalho global para começar a falar.
Fale claramente para enviar comandos ou digitar texto.
