# 🧠 الوضع الهجين SL5 Aura: تكامل LLM والحافظة المحلية

                                               **الحالة:** مستقرة
      **التقنية:** Ollama (Llama 3.2) + بنية جسر الملفات
                **الخصوصية:** 100% غير متصل بالإنترنت

        ## المفهوم: "المهندس المعماري والمتدرب"

تقليديًا، تعتمد Aura على القواعد الحتمية (RegEx) – السريعة والدقيقة. هذا هو **"المهندس المعماري"**.
يعمل **المكوّن الإضافي LLM المحلي** بمثابة **"المتدرب"**: فهو يتعامل مع الطلبات الغامضة، ويلخص النصوص، ويجيب على الأسئلة العامة.

             ## 🛠 الهندسة المعمارية: جسر الحافظة

نظرًا للقيود الأمنية في Linux (Wayland/X11)، غالبًا ما لا تتمكن عمليات الخلفية (مثل Aura) من الوصول إلى الحافظة مباشرةً. لقد قمنا بحل هذه المشكلة باستخدام **هندسة الجسر**:

1. **الموفر (جلسة المستخدم):** يتم تشغيل برنامج نصي صغير (`clipboard_bridge.sh`) في جلسة المستخدم. يقوم بمراقبة الحافظة ويعكس محتواها إلى ملف مؤقت (`/tmp/aura_clipboard.txt`).
2. **المستهلك (Aura):** يقرأ البرنامج المساعد Python هذا الملف. وبما أن الوصول إلى الملفات هو أمر عالمي، فسيتم تجاوز مشكلات الأذونات.

                                              ## 🚀 دليل الإعداد

                                        ### 1. قم بتثبيت أولاما
```bash
sudo pacman -S ollama xclip wl-clipboard
sudo systemctl enable --now ollama
ollama run llama3.2
```

                     ### 2. إعداد البرنامج النصي للجسر
   أنشئ `~/clipboard_bridge.sh` واجعله قابلاً للتنفيذ:

```bash
#!/bin/bash
# Mirrors clipboard to a file in RAM
FILE="/tmp/aura_clipboard.txt"
while true; do
    if command -v wl-paste &> /dev/null; then
        wl-paste --no-newline > "$FILE" 2>/dev/null
    else
        xclip -selection clipboard -o > "$FILE" 2>/dev/null
    fi
    sleep 1.5
done
```

**هام:** أضف هذا البرنامج النصي إلى نظام التشغيل التلقائي لديك!

            ### 3. منطق البرنامج المساعد (`ask_ollama.py`)

يوجد البرنامج النصي في `config/maps/plugins/z_fallback_llm/de-DE/`.
* **المشغل:** يكتشف كلمات مثل "Computer"، و"Aura"، و"Clipboard"، و"Summary".
* **الذاكرة:** تحتفظ بـ "conversation_history.json" لتذكر السياق (على سبيل المثال، "ماذا سألت للتو؟").
* **الهندسة السريعة:** تعطي الأولوية لبيانات الحافظة الحالية على سياق المحادثة التاريخية لمنع الهلوسة.

                                        ## 📝 أمثلة الاستخدام

                                                  1. **تلخيص النص:**
* *الإجراء:* انسخ رسالة بريد إلكتروني طويلة أو نص موقع ويب (Ctrl+C).
* *أمر صوتي:* "أيها الكمبيوتر، قم بتلخيص النص الموجود في الحافظة."

                                        2. **الترجمة/التحليل:**
                        * *الإجراء:* انسخ مقتطف الشفرة.
* *أمر صوتي:* "أيها الكمبيوتر، ما وظيفة الكود الموجود في الحافظة؟"

                                          3. **الدردشة العامة:**
* *أمر صوتي:* "أيها الكمبيوتر، أخبرني نكتة عن المبرمجين."

                                     4. **إعادة ضبط الذاكرة:**
* *الأمر الصوتي:* "أيها الكمبيوتر، انسَ كل شيء." (يمسح سجل JSON).
