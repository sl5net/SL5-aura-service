# 💡 功能亮点：持久插件缓存

## 💾 缓存以提高性能、可靠性和网络效率

该系统现在提供了一个中央持久缓存机制（“simple_plugin_cache”），旨在减少外部网络负载（例如天气、翻译 API）并加速执行。缓存是**持久的**，这意味着条目在服务重新启动后仍然存在。

**目标：** 减少网络流量，防止 API 滥用，并在外部服务中断期间提供快速回退响应。

#### B. 缓存键（`key_args`）要求

* `key_args` 必须是一个**元组**，包含定义结果的所有变量（例如，`(city, language, unit_system)`）。
* 缓存机制自动将常见的非JSON可序列化的Python对象（例如**`pathlib.Path`**）转换为字符串以用于密钥生成。
