# 无与伦比的训练插件 (`1_collect_unmatched_training`)

＃＃ 目的

该插件自动收集无法识别的语音输入并将其添加
作为模糊映射正则表达式的新变体。这使得系统能够“自我训练”
随着时间的推移，通过从无与伦比的识别结果中学习。

## 它是如何工作的

1. 当没有其他规则匹配时，会触发“COLLECT_UNMATCHED”包罗万象的规则。
2. 通过“on_match_exec”使用匹配的文本调用“collect_unmatched.py”。
3. 调用`FUZZY_MAP_pre.py`中的正则表达式会自动扩展。

## 禁用插件

当您收集了足够的训练数据后，可以通过以下任一方式禁用：

- 注释掉包罗万象的规则
- 使用无效名称重命名文件夹（例如添加空格）
- 从“maps”目录中删除插件文件夹
