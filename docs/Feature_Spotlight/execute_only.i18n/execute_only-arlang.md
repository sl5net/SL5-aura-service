# سمة القاعدة: `التنفيذ_فقط` (تجريبي، 7.7.\'26 الثلاثاء)

السمة `execute_only` هي خيار تكوين تجريبي مصمم للقواعد التي تقوم فقط بتشغيل البرامج النصية الخارجية دون تعديل نص الإدخال أو استبداله.

                                                                  ## ملخص
- **النوع:** `منطقي` (على سبيل المثال، `صحيح` أو `خطأ`)
- **حالة الاستخدام الأساسية:** تُستخدم عادةً مع `on_match_exec` لتشغيل البرامج النصية الخارجية.

                               ## كيف يعمل والسلوك الحالي
- **تحسين السرعة:** (بضعة ميلي ثانية فقط) لتجاوز إجراءات المعالجة اللاحقة للنص واستبدال النص، مما يؤدي إلى تسريع التنفيذ الفوري للإجراء الذي تم تشغيله.
- **لا يوجد استثناء/تأثير جانبي شامل:** ضبط `تنفيذ_فقط` على `صحيح` لا **لا** يمنع قواعد المطابقة الأخرى من تقييم نفس النص المُدخل.
- **إيقاف التدفق:** إذا كنت بحاجة إلى إيقاف القواعد اللاحقة من معالجة نفس نص الإدخال، فيجب عليك حاليًا إنهاء تدفق التنفيذ يدويًا (على سبيل المثال، عن طريق طرح استثناء في نهاية البرنامج النصي الذي تم تشغيله أو معالج مجموعة القواعد).

                                                   ## مثال للتكوين

```python
# EXAMPLE: gather metal
('gather metal',
 r'^(gather\s*)?(met\w+|mat\w+|metall|mit|zitat|metal|matcha|günther)$',
 85,
 {
     'flags': re.IGNORECASE,
     'only_in_windows': ['0ad', '0AD', '0 a.d.', '0 a.d'],
     'on_match_exec': [CONFIG_DIR / '..' / '0ad_actions.py'],
     'execute_only': True, # Experimental: Fast execution, does not halt the rule-chain.
 }),
```
