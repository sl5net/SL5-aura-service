# Globalny skrót CopyQ do Super+s

## Organizować coś
1. Otwórz CopyQ i naciśnij `F6`.
2. Kliknij Dodaj -> Nowe polecenie.
3. Wklej ten przykład JavaScript search_rules.sh działa w systemie Linux:
__KOD_BLOKU_0__

4. Ustaw skrót globalny na „Super+s”.
5. Kliknij OK.


run_rule.sh tylko dla Linuksa:
__KOD_BLOKU_1__


4. Ustaw skrót globalny na „Super+y”.
5. Kliknij OK.

## Zgodność
- Testowano na systemie Manjaro Linux (KDE Plasma, X11).
- Nie testowano w systemie Windows ani macOS.
