# 规则属性：`only_in_windows` 和 `exclude_windows`

这两个属性控制**允许在哪些活动窗口中触发规则**。
它们在规则的“选项”字典中定义，并接受**正则表达式模式列表**
与当前活动窗口标题（`_active_window_title`）匹配。

---

## `仅在 Windows 中`

**仅当**活动窗口标题与**至少一个**给定模式匹配时，该规则才会触发。
所有其他窗口都将被忽略。

**用例：** 将规则限制到特定应用程序。


> **仅**当 Firefox 或 Chromium 是活动窗口时，该规则才会触发。

---

## `排除窗口`

**除非**活动窗口标题与给定模式中的**至少一个**匹配，否则规则将触发。
匹配的窗口被跳过。

**用例：** 禁用特定应用程序的规则。

示例

__代码_块_0__



匹配**不区分大小写**并使用 Python **正则表达式**。

---

＃＃ 概括

|属性|当...时触发
|--------------------------------|--------------------------------------------------------|
| `仅在 Windows 中` |窗口标题 **匹配** 模式之一 |
| `排除窗口` |窗口标题**不匹配**任何模式|
