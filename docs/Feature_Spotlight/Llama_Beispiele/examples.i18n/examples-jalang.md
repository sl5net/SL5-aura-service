Hier sind **5 konkrete Szenarien**, die du mit deinem aktuellen をセットアップしてください:

### 1. Der "Text-Polierer" (私のお気に入り)
シュネルと構造を明確にします。 LLM kann das glätten、電子メールをよろしくお願いします。
* **Du sagst:** *「Aura formuliere das höflich: Ich hab keine Zeit für das Meeting, verschieben.」*
* **LLM macht:** *「Sehr geehrte Damen und Herren, leider kann ich den geplanten Termin nicht wahrnehmen. Wäre eine Verschiebung möglich?」*
* **Warum:** RegEx kann keine Grammatik。 LLM シェーンです。

### 2.「Linux-Spickzettel」
「man ページ」を読んで、詳細を確認してください。 Das Modell はコードのトレーニングに使用されます。
* **Du sagst:** *「Linux に関する Aura の検索結果は 100 MB ですか?」*
* **LLM マッチ:** `find / -type f -size +100M`
* **Warum:** フローに沿って、フェンスを越えてください。

### 3. Strukturierte Daten extrahieren (Deine "Restaurant-Idee")
**Königsdisziplin** が死ぬことです。 LLM については、「ハート」の「weiche」 Sprache を、オーラ スクリプト umzuwandeln で確認してください。
* **Du sagst:** *「オーラ ベストテル ピザ サラミ アバー シャーフ アンド オーネ ツヴィーベルン。」*
* **LLM (JSON プロンプトを含む):** `{"製品": "ピザ サラミ", "mods": ["+scharf", "-Zwiebeln"]}`
* **Vorteil:** Python-Skript muss nicht tausende Varianten von "ohne"、"kein"、"bitte nicht" parsen。 LLM の究極。

### 4. オフライン ツーサンメンファッスンゲン
Wenn du einen langen Zwischenablage hast (oder diktiert hast) のテキスト:
* **Du sagst:** *「Aura fasse den folgenden Text in drei Stichpunkten zusammen: [Text]」*
* **LLM macht:** Liefert dir sofort die Essenz。

### 5.「クリエイティブフォールバック」
Wenn dir ein Wort fehlt oder du eine Idee brauchst.
* **Du sagst:** *「Aura nenne mir 5 Synonyme für 'Entwicklung'.」*
* **LLM macht:** *「Fortschritt、Evolution、Entfaltung、Genese、Ausarbeitung。」*
* **LLM macht:** *「Entwicklung、Fortschritt、Innovation、Fortbildung、Verbesserung。**」
