# उपयोगकर्ता-विशिष्ट वॉयस कमांड

ऑरा आपको कस्टम कमांड को परिभाषित करने की अनुमति देता है जो **केवल आपके लिए सक्रिय हैं** (या विशिष्ट टीम के सदस्य)। यह व्यक्तिगत शॉर्टकट या प्रयोगात्मक सुविधाओं को अन्य उपयोगकर्ताओं द्वारा ट्रिगर होने से रोकता है।
