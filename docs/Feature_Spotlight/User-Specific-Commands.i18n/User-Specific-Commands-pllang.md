# Polecenia głosowe specyficzne dla użytkownika

Aura umożliwia definiowanie niestandardowych poleceń, które są **aktywne tylko dla Ciebie** (lub określonych członków zespołu). Zapobiega to uruchamianiu osobistych skrótów lub funkcji eksperymentalnych przez innych użytkowników.

## Organizować coś

Możesz zastosować te reguły w dowolnym pliku mapowania, takim jak `FUZZY_MAP_pre.py` (surowe dane wejściowe) lub `FUZZY_MAP.py` (po korekcie).

Plik docelowy: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`

### Przykład kodu

Dodaj ten blok na końcu pliku:

__KOD_BLOKU_0__
