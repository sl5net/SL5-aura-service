# Benutzerspezifische Sprachbefehle

Mit Aura können Sie benutzerdefinierte Befehle definieren, die **nur für Sie aktiv** (oder bestimmte Teammitglieder) sind. Dadurch wird verhindert, dass andere Benutzer persönliche Verknüpfungen oder experimentelle Funktionen auslösen.

## Aufstellen

Sie können diese Regeln in jeder Zuordnungsdatei anwenden, z. B. „FUZZY_MAP_pre.py“ (Roheingabe) oder „FUZZY_MAP.py“ (nach Korrektur).

Zieldatei: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`

### Codebeispiel

Fügen Sie diesen Block am Ende der Datei hinzu:

__CODE_BLOCK_0__
