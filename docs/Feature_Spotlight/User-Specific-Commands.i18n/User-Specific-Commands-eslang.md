# Comandos de voz específicos del usuario

Aura te permite definir comandos personalizados que **solo están activos para ti** (o para miembros específicos del equipo). Esto evita que otros usuarios activen atajos personales o funciones experimentales.

## Configuración

Puede aplicar estas reglas en cualquier archivo de mapeo, como `FUZZY_MAP_pre.py` (entrada sin formato) o `FUZZY_MAP.py` (después de la corrección).

Archivo de destino: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`
