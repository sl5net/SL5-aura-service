# ユーザー固有の音声コマンド

Aura を使用すると、**自分 (または特定のチームメンバー) に対してのみ有効**なカスタム コマンドを定義できます。これにより、個人のショートカットや実験的な機能が他のユーザーによってトリガーされるのを防ぎます。

＃＃ 設定

これらのルールは、`FUZZY_MAP_pre.py` (生の入力) または `FUZZY_MAP.py` (修正後) などのマッピング ファイルに適用できます。

対象ファイル: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`
