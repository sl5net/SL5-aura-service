# 사용자별 음성 명령

Aura를 사용하면 **본인(또는 특정 팀 구성원)에게만 활성화되는** 사용자 지정 명령을 정의할 수 있습니다. 이렇게 하면 다른 사용자가 개인 바로가기나 실험적 기능을 실행하는 것을 방지할 수 있습니다.

## 설정

`FUZZY_MAP_pre.py`(원시 입력) 또는 `FUZZY_MAP.py`(수정 후)와 같은 모든 매핑 파일에 이러한 규칙을 적용할 수 있습니다.

대상 파일: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`
