# الأوامر الصوتية الخاصة بالمستخدم

تتيح لك Aura تحديد الأوامر المخصصة **النشطة لك فقط** (أو لأعضاء فريق محددين). وهذا يمنع تشغيل الاختصارات الشخصية أو الميزات التجريبية بواسطة مستخدمين آخرين.

                                                                  ## يثبت

يمكنك تطبيق هذه القواعد في أي ملف تعيين، مثل `FUZZY_MAP_pre.py` (الإدخال الأولي) أو `FUZZY_MAP.py` (بعد التصحيح).

الملف الهدف: `config/maps/plugins/internals/de-DE/FUZZY_MAP_pre.py`
