# سير العمل المضغوط تلقائيًا والوثائق المضمنة

                                                                ## مفهوم
يقوم SL5 Aura تلقائيًا بمراقبة المجلدات التي تبدأ بـ `_` (على سبيل المثال، `_my_application`). عند اكتشاف التغييرات، يقوم Aura تلقائيًا بضغط المجلد في أرشيف مضغوط.

                                               ** القيد الحاسم: **
يستمع نظام "Hot-Reload" والمراقبة الخاص بـ Aura بشكل خاص إلى التغييرات في **ملفات Python الصالحة**. لن يؤدي تحديث ملف نصي بسيط (`.txt`) إلى تشغيل عملية الضغط التلقائي.

                               ## نمط "المستندات المضمنة".
لتضمين تعليمات للمستلمين غير التقنيين (على سبيل المثال، الموارد البشرية والعملاء) مع التأكد من اكتشاف Aura للتغيير وتحديث الملف المضغوط، نستخدم **Python Docstring File**.

يعد هذا الملف من الناحية الفنية نصًا صالحًا لـ Python (يُرضي محلل Aura) ولكنه يظهر بصريًا كمستند نصي قياسي للمستخدم.

                                                               ### تطبيق
قم بإنشاء ملف باسم `README_AUTOZIP.py` داخل المجلد المراقب.

                                                 **دليل الأسلوب:**
1. استخدم `# Documentation` كسطر أول (بدلاً من اسم البرنامج النصي الفني) للترحيب بك.
      2. استخدم اقتباسًا ثلاثيًا (`"""`) للمحتوى.
                                   3. لا يوجد رمز آخر مطلوب.
