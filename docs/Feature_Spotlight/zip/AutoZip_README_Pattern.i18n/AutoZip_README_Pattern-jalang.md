# 自動圧縮ワークフローと埋め込みドキュメント

＃＃ コンセプト
SL5 Aura は、「_」で始まるフォルダー (例: _my_application`) を自動的に監視します。変更が検出されると、Aura は自動的にフォルダーを zip アーカイブに圧縮します。

**重要な制約:**
Aura の「ホットリロード」および監視システムは、**有効な Python ファイル**の変更を特にリッスンします。単純なテキスト ファイル (`.txt`) の更新では、自動圧縮プロセスは**トリガーされません**。

## 「埋め込みドキュメント」パターン
Aura が変更を検出して zip を更新できるようにしながら、技術者以外の受信者 (人事、クライアントなど) への指示を含めるために、**Python Docstring ファイル** を使用します。

このファイルは技術的には有効な Python スクリプト (Aura のパーサーを満たす) ですが、ユーザーには視覚的に標準のテキストドキュメントとして表示されます。

＃＃＃ 実装
監視対象フォルダー内に「README_AUTOZIP.py」という名前のファイルを作成します。

**スタイルガイド:**
1. 歓迎の意を示すために、最初の行 (技術的なスクリプト名の代わりに) として「# Documentation」を使用します。
2. コンテンツには三重引用符 (`"""`) のドキュメント文字列を使用します。
3. 他のコードは必要ありません。
