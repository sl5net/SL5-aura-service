# 自动压缩工作流程和嵌入式文档

＃＃ 概念
SL5 Aura 自动监控以“_”开头的文件夹（例如“_my_application”）。当检测到更改时，Aura 会自动将文件夹压缩为 zip 存档。

**关键约束：**
Aura 的“热重载”和监控系统专门监听**有效 Python 文件**中的更改。简单的文本文件（`.txt`）更新将**不会**触发自动压缩过程。

## “嵌入式文档”模式
为了向非技术接收者（例如 HR、客户）提供说明，同时确保 Aura 检测到更改并更新 zip，我们使用 **Python 文档字符串文件**。

从技术上讲，该文件是一个有效的 Python 脚本（满足 Aura 的解析器），但在视觉上对用户显示为标准文本文档。

＃＃＃ 执行
在受监控的文件夹中创建一个名为“README_AUTOZIP.py”的文件。

**风格指南：**
1. 使用 `# Documentation` 作为第一行（而不是技术脚本名称）以表示欢迎。
2. 对内容使用三引号 (`"""`) 文档字符串。
3. 不需要其他代码。
