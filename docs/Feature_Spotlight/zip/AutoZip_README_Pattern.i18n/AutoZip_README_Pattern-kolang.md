# 자동 압축 작업 흐름 및 포함된 문서

## 개념
SL5 Aura는 `_`로 시작하는 폴더(예: `_my_application`)를 자동으로 모니터링합니다. 변경 사항이 감지되면 Aura는 자동으로 폴더를 zip 아카이브로 압축합니다.

**중요한 제약:**
Aura의 "핫 리로드" 및 모니터링 시스템은 **유효한 Python 파일**의 변경 사항을 구체적으로 수신합니다. 간단한 텍스트 파일(`.txt`) 업데이트는 자동 압축 프로세스를 실행하지 **않습니다**.

## "내장 문서" 패턴
Aura가 변경 사항을 감지하고 zip을 업데이트하는 동시에 기술적 지식이 없는 수신자(예: HR, 클라이언트)를 위한 지침을 포함하기 위해 우리는 **Python Docstring File**을 사용합니다.

이 파일은 기술적으로 유효한 Python 스크립트(Aura의 파서 만족)이지만 사용자에게는 시각적으로 표준 텍스트 문서로 나타납니다.

### 구현
모니터링되는 폴더 안에 `README_AUTOZIP.py`라는 파일을 만듭니다.

**스타일 가이드:**
1. 첫 번째 줄에 `# Documentation`을 사용하여(기술 스크립트 이름 대신) 환영합니다.
2. 콘텐츠에 삼중따옴표(`"""`) Docstring을 사용하십시오.
3. 다른 코드는 필요하지 않습니다.
