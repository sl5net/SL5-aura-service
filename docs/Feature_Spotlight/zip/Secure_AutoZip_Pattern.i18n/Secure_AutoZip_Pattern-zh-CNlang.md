# 安全的自动压缩和嵌入式文档

＃＃ 概念
SL5 Aura 监控以“_”开头的私人文件夹（例如“_my_confidential_data”）。
当检测到更改时，Aura 会自动创建一个**加密的** zip 存档。

## 关键先决条件：加密密钥
**加密是强制性的。** 自动压缩过程严格要求目录层次结构（当前或父文件夹）中存在密码文件。

* **文件要求：** 密码文件必须以点“.”开头（例如“.archive_pass”、“.secret”）。
* **行为：** 如果未找到带有密码的点文件，则 zip 进程将被 **阻止**。这种故障保护确保不会打包任何未加密的数据。

## “嵌入式文档”模式
由于 Aura 的热重载系统会侦听 **有效的 Python 文件**，因此更新简单的“.txt”自述文件不会触发重新压缩。

要在确保触发器触发的同时包含针对收件人的说明（例如“如何解压缩”），请使用 **Python 文档字符串文件**。

＃＃＃ 执行
在受监控的文件夹中创建一个名为“README_AUTOZIP.py”的文件。
