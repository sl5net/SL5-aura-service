# 안전한 자동 압축 및 내장 문서

## 개념
SL5 Aura는 `_`(예: `_my_confidential_data`)로 시작하는 개인 폴더를 모니터링합니다.
변경 사항이 감지되면 Aura는 **암호화된** zip 아카이브를 자동으로 생성합니다.

## 중요 전제 조건: 암호화 키
**암호화는 필수입니다.** 자동 압축 프로세스에서는 디렉터리 계층 구조(현재 또는 상위 폴더)에 비밀번호 파일이 있어야 합니다.

* **파일 요구 사항:** 비밀번호 파일은 점 `.`으로 시작해야 합니다(예: `.archive_pass`, `.secret`).
* **동작:** 비밀번호가 포함된 도트 파일이 발견되지 않으면 zip 프로세스가 **차단됩니다**. 이 안전 장치는 암호화되지 않은 데이터가 패키지화되지 않도록 보장합니다.

## "내장 문서" 패턴
Aura의 핫 리로드 시스템은 **유효한 Python 파일**을 수신하므로 간단한 `.txt` readme를 업데이트해도 재압축이 실행되지 않습니다.

트리거가 실행되는지 확인하는 동안 수신자를 위한 지침(예: "압축 해제 방법")을 포함하려면 **Python Docstring File**을 사용하세요.
