# 安全な自動圧縮と埋め込みドキュメント

＃＃ コンセプト
SL5 Aura は、「_」で始まるプライベートフォルダー (例: `_my_confidential_data`) を監視します。
変更が検出されると、Aura は **暗号化された** zip アーカイブを自動的に作成します。

## 重要な前提条件: 暗号化キー
**暗号化は必須です。** 自動圧縮プロセスでは、ディレクトリ階層 (現在のフォルダーまたは親フォルダー) にパスワード ファイルが存在することが厳密に必要です。

* **ファイル要件:** パスワード ファイルはドット `.` で始まる必要があります (例: `.archive_pass`、`.secret`)。
* **動作:** パスワード付きのドットファイルが見つからない場合、zip プロセスは **ブロックされます**。このフェイルセーフにより、暗号化されていないデータがパッケージ化されることがなくなります。

## 「埋め込みドキュメント」パターン
Aura のホットリロード システムは **有効な Python ファイル**をリッスンするため、単純な `.txt` Readme を更新しても再圧縮はトリガーされません。

トリガーを確実に起動しながら、受信者への指示 (「解凍方法」など) を含めるには、**Python Docstring ファイル** を使用します。

監視対象フォルダー内に「README_AUTOZIP.py」という名前のファイルを作成します。
