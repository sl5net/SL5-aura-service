# 윈도우에서 시작하기

## 1단계: 설치 프로그램 실행
`setup/windows11_setup_with_ahk_copyq.bat`를 두 번 클릭합니다.
- 메시지가 나타나면 마우스 오른쪽 버튼을 클릭하고 → "관리자 권한으로 실행"을 클릭합니다.
- 스크립트는 Python, AutoHotkey v2, CopyQ를 설치하고 음성 모델(~4GB)을 다운로드합니다.
- 대략 8~10분 정도 소요됩니다.

## 2단계: Aura 시작
프로젝트 폴더에서 `start_aura.bat`를 더블클릭하세요.
시작 소리가 들려야 합니다. Aura가 준비되었습니다.

**아무 일도 일어나지 않았나요?** 로그를 확인하세요.
로그\aura_engine.log

## 3단계: 단축키 구성
설치 프로그램은 CopyQ를 자동으로 설치합니다. 받아쓰기를 실행하려면 다음 안내를 따르세요.
1. CopyQ 열기 → 명령 → 명령 추가
2. 명령을 다음과 같이 설정합니다.
cmd /c 에코. > C:\tmp\sl5_record.trigger
3. 전역 단축키 지정(예: `F9`)

## 4단계: 첫 번째 받아쓰기
1. 텍스트 필드를 클릭하세요.
2. 단축키를 누르세요. "듣는 중..." 알림이 나올 때까지 기다립니다.
3. "Hello World"라고 말하세요.
4. 단축키를 다시 누르면 텍스트가 나타납니다.

## 5단계: 음성 명령 찾기
**"Aura Search"**라고 말합니다. 사용 가능한 모든 규칙이 포함된 창이 열립니다.

## 문제 해결
| 증상 | 수정 |
|---|---|
| 시작음이 들리지 않음 | `log\aura_engine.log` 확인 |
| 단축키는 아무 것도 하지 않습니다 | `C:\tmp\sl5_record.trigger`가 생성되었는지 확인 |
| 텍스트가 입력되지 않았습니다 | 작업 관리자에서 `type_watcher.ahk`가 실행 중인지 확인 |
| 시작 시 충돌 | 관리자로 설치를 다시 실행 |
