# الشروع في العمل على نظام التشغيل Windows

             ## الخطوة 1: قم بتشغيل برنامج الإعداد
انقر نقرًا مزدوجًا فوق "setup/windows11_setup_with_ahk_copyq.bat".
- انقر بزر الماوس الأيمن → "تشغيل كمسؤول" إذا طُلب منك ذلك.
- يقوم البرنامج النصي بتثبيت Python وAutoHotkey v2 وCopyQ وتنزيل النماذج الصوتية (~4 جيجابايت).
                            - يستغرق ذلك حوالي 8-10 دقائق.

                                     ## الخطوة 2: ابدأ الهالة
انقر نقرًا مزدوجًا فوق `start_aura.bat` في مجلد المشروع.
         يجب أن تسمع صوت بدء التشغيل - Aura جاهزة.

                         **لم يحدث شيء؟** تحقق من السجل:
                                                       سجل\aura_engine.log

## الخطوة 3: قم بتكوين مفتاح التشغيل السريع الخاص بك
يقوم الإعداد بتثبيت CopyQ تلقائيًا. لتفعيل الإملاء:
                   1. افتح CopyQ → الأوامر → إضافة أمر
                                               2. اضبط الأمر على:
                              كمد / ج صدى. > C:\tmp\sl5_record.trigger
                       3. قم بتعيين اختصار عام (مثل `F9`)

                                 ## الخطوة 4: الإملاء الأول
                                        1. انقر فوق أي حقل نصي
2. اضغط على مفتاح التشغيل السريع - انتظر إشعار "الاستماع...".
                                        3. قل "مرحبًا بالعالم"
4. اضغط على مفتاح التشغيل السريع مرة أخرى - يظهر النص

               ## الخطوة 5: ابحث عن الأوامر الصوتية
قل: **"Aura Search"** — تفتح نافذة تحتوي على جميع القواعد المتاحة.

                            ## استكشاف الأخطاء وإصلاحها
                                              | العَرَض | إصلاح |
                                                                    |---|---|
| لا يوجد صوت بدء التشغيل | تحقق من `log\aura_engine.log` |
| مفتاح التشغيل السريع لا يفعل شيئًا | تحقق من إنشاء `C:\tmp\sl5_record.trigger` |
| النص غير مكتوب | تحقق مما إذا كان `type_watcher.ahk` قيد التشغيل في إدارة المهام |
| تحطم عند البداية | قم بتشغيل برنامج الإعداد مرة أخرى كمسؤول |
