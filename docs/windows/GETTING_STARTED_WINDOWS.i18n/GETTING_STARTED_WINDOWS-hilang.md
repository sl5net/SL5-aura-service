# विंडोज़ पर आरंभ करना

## चरण 1: सेटअप चलाएँ
`setup/windows11_setup_with_ahk_copyq.bat` पर डबल-क्लिक करें।
- संकेत मिलने पर राइट-क्लिक करें → "व्यवस्थापक के रूप में चलाएँ"।
- स्क्रिप्ट Python, AutoHotkey v2, CopyQ इंस्टॉल करती है और वॉयस मॉडल (~4GB) डाउनलोड करती है।
- इसमें लगभग 8-10 मिनट का समय लगता है।

## चरण 2: आभा प्रारंभ करें
प्रोजेक्ट फ़ोल्डर में `start_aura.bat` पर डबल-क्लिक करें।
आपको एक स्टार्टअप ध्वनि सुननी चाहिए - ऑरा तैयार है।

**कुछ नहीं हुआ?** लॉग जांचें:
लॉग\aura_engine.log

## चरण 3: अपनी हॉटकी कॉन्फ़िगर करें
सेटअप स्वचालित रूप से CopyQ इंस्टॉल करता है। श्रुतलेख ट्रिगर करने के लिए:
1. CopyQ → Commands → Command जोड़ें खोलें
2. कमांड को इस पर सेट करें:
सीएमडी/सी इको. > सी:\tmp\sl5_record.trigger
3. एक वैश्विक शॉर्टकट निर्दिष्ट करें (उदा. `F9`)

## चरण 4: पहला श्रुतलेख
1. किसी भी टेक्स्ट फ़ील्ड पर क्लिक करें
2. अपनी हॉटकी दबाएं - "सुनना..." अधिसूचना की प्रतीक्षा करें
3. "हैलो वर्ल्ड" कहें
4. हॉटकी को दोबारा दबाएं - टेक्स्ट प्रकट होता है

## चरण 5: वॉयस कमांड ढूंढें
कहें: **"आभा खोज"** - सभी उपलब्ध नियमों के साथ एक विंडो खुलती है।

## समस्या निवारण
| लक्षण | ठीक करें |
|---|---|
| कोई स्टार्टअप ध्वनि नहीं | `log\aura_engine.log` जांचें |
| हॉटकी कुछ नहीं करती | जांचें कि क्या `C:\tmp\sl5_record.trigger` बनाया गया है |
| पाठ टाइप नहीं हुआ | जांचें कि क्या `type_watcher.ahk` टास्क मैनेजर में चल रहा है |
| प्रारंभ पर दुर्घटना | व्यवस्थापक के रूप में फिर से सेटअप चलाएँ |
