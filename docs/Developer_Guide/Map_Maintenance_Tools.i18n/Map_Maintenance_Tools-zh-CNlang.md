### 工作流程
1. **创建或编辑**映射文件（例如，添加新规则）。
2. **运行**标记器脚本。
3. **交互模式：**
- 该脚本将向您显示生成的建议。
- 按“ENTER”接受它。
- 输入“s”跳过。
- 如果您想跳过生成失败的所有剩余模式，请输入“sa”（跳过全部）。
4. **提交**更改。

> **注意：** 该脚本会忽略现有的 `# Examples:` 标签，因此可以安全地重复运行。
