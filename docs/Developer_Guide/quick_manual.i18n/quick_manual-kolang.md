## 고급 규칙 속성

표준 필드 외에도 특수 옵션을 사용하여 규칙을 향상할 수 있습니다.

### `only_in_windows`(창 제목 필터)
이름에도 불구하고 이 속성은 **OS 독립적**입니다. 현재 활성 창의 제목을 기준으로 규칙을 필터링합니다.

* **기능:** 활성 창 제목이 제공된 패턴(Regex) 중 하나와 일치하는 경우에만 규칙이 처리됩니다.
*   **예:**
    ```python
    (
        '|', 
        r'\b(pipe|symbol)\b', 
        75, 
        {'only_in_windows': ['Terminal', 'Console', 'iTerm']}
    ),
    ```
*이 경우 교체는 사용자가 터미널 창 내에서 작업하는 경우에만 발생합니다.*

### `on_match_exec`(스크립트 실행)
규칙이 일치할 때 외부 Python 스크립트를 트리거할 수 있습니다.

* **구문:** `'on_match_exec': [CONFIG_DIR / 'script.py']`
* **사용 사례:** API 호출, 파일 시스템 작업 또는 동적 콘텐츠 생성과 같은 복잡한 작업에 적합합니다.
