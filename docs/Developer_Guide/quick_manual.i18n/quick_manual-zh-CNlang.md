## 高级规则属性

除了标准字段之外，还可以使用特殊选项来增强规则：

### `only_in_windows`（窗口标题过滤器）
尽管有其名称，但该属性是**独立于操作系统的**。它根据当前活动窗口的标题过滤规则。

* **功能：** 仅当活动窗口标题与提供的模式（正则表达式）之一匹配时，才会处理该规则。
*   **例子：**
__代码_块_0__
*在这种情况下，仅当用户在终端窗口内工作时才会发生替换。*

### `on_match_exec`（脚本执行）
允许在规则匹配时触发外部 Python 脚本。

* **语法：** `'on_match_exec': [CONFIG_DIR / 'script.py']`
* **用例：** 非常适合复杂的操作，例如 API 调用、文件系统任务或生成动态内容。
