## سمات القاعدة المتقدمة

بالإضافة إلى الحقول القياسية، يمكن تعزيز القواعد بخيارات خاصة:

                   ### `only_in_windows` (مرشح عنوان النافذة)
على الرغم من اسمها، فإن هذه السمة **مستقلة عن نظام التشغيل**. يقوم بتصفية القواعد بناءً على عنوان النافذة النشطة حاليًا.

* **الوظيفة:** تتم معالجة القاعدة فقط إذا كان عنوان النافذة النشطة يتطابق مع أحد الأنماط المتوفرة (Regex).
                                                            *   **مثال:**
    ```python
    (
        '|', 
        r'\b(pipe|symbol)\b', 
        75, 
        {'only_in_windows': ['Terminal', 'Console', 'iTerm']}
    ),
    ```
*في هذه الحالة، يتم الاستبدال فقط إذا كان المستخدم يعمل داخل نافذة طرفية.*

                 ### `on_match_exec` (تنفيذ البرنامج النصي)
يسمح بتشغيل نصوص بايثون الخارجية عندما تتطابق القاعدة.

  * **بناء الجملة:** `\'on_match_exec': [CONFIG_DIR / 'script.py']`
* **حالة الاستخدام:** مثالية للإجراءات المعقدة مثل استدعاءات واجهة برمجة التطبيقات (API)، أو مهام نظام الملفات، أو إنشاء محتوى ديناميكي.
