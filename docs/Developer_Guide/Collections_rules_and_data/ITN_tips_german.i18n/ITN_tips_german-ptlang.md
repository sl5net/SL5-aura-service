der Fachbegriff ist **Normalização de texto inverso (ITN)**.

Quando você danach assim, encontre o riesige Sammlungen von Regeln und Daten.

Aqui estão os melhores recursos, um mapa para ser preenchido, mas todos os que estão disponíveis:

### 1. ITN-Regel-Sammlungen (Der "Padrão Ouro")
* **[itnpy](https://github.com/barseghyanartur/itnpy):** Uma ferramenta Python determinística genau para esta ferramenta. É um CSV-Dateien nutzt, um texto atualizado em um tempo registrado (Zahlen, Währungen, Daten) para verwandeln. Os CSVs podem ser copiados rapidamente em 1:1 no seu mapa.

* **[NVIDIA NeMo ITN](https://github.com/NVIDIA/NeMo):** Sehr mächtig. Você tem uma série de datas gramaticais para todos os idiomas. Dort findest du Listen für Maßeinheiten, Titel und Datumsformate.

### 2. Datas para pontuação e maiúsculas e minúsculas
* **[Vosk recasepunc](https://github.com/benob/recasepunc):** Esta é a ferramenta padrão para Vosk. É simplesmente um modelo, mas no Quellcode é encontrado com frequência para ouvir para Abkürzungen e Eigennamen, o homem extrahieren pode.

* **[Google Text Normalization Dataset](https://github.com/rwsproat/text-normalization-data):** Ein riesiger Datensatz (für eine Kaggle-Challenge erstellt), der Millionen von Beispielen enthält, wie gesprochene Sprache (`falado`) em geschriebene (`escrito`) umgewandelt wird.

### 3. Biblioteca "Diktat-Helfer"
* **[num2words](https://github.com/savoirfairelinux/num2words):** Se o mapeamento de Zahlen for concluído, você poderá dir aqui o Listen für „eins“ bis „eine Million“ gerieren lassen.
