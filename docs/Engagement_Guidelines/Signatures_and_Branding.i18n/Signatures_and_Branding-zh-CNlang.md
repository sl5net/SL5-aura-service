# 沟通指南：签名和品牌

为了帮助传播 **Aura** 的意识而不具有侵入性，我们通过聊天签名使用“被动品牌”策略。这使得用户可以轻松找到项目，同时保持对话集中在内容上。

## 1. 光环签名
在聊天中使用翻译或自动化功能时，Aura 可以在您的消息中附加一个小签名。

**推荐格式：**
> `🗣SL5net ⟫ 光环`

### 符号细分：
* **`🗣`（讲话头）：** 此消息已处理或翻译的视觉指示器。它表示“交流/语言”。
* **`SL5net`:** 唯一的命名空间。这对于可搜索性至关重要。
* **`⟫`（双右括号）：** 技术“管道”符号，看起来像 CLI/Shell 运算符，强调这是一个技术工具。
* **`Aura`:** 项目名称。

---

## 2. 为什么使用“SL5net Aura”而不是 Link？
许多聊天平台（Matrix、Discord、Telegram 等）对 URL 或类似域的结构都有严格的过滤器。
* **可搜索性：** 在 Google 或 GitHub 上搜索“SL5net Aura”，找到存储库的成功率为 100%。
* **过滤安全：** 通过使用独特的搜索词而不是“.com”链接，我们可以避免被自动审核员标记为“垃圾邮件”或“广告机器人”。
* **低摩擦：** 易于阅读并且易于在搜索栏中输入。

---

## 3. 签名礼仪（常见问题解答）
如何有效地使用签名而不打扰您的聊天伙伴：

### 问：我应该在每条消息上签名吗？
**答：** 不。在快节奏的聊天中，恒定的签名可能会被视为垃圾邮件。
* **最佳实践：** 仅启用新对话中第一条消息或特定“高价值”翻译消息的签名。
* **配置：** 在私人或高度正式的商务会议中，使用配置中的“#signatur”开关将其关闭。

### 问：为什么不使用盲文或复杂的 Unicode 字符？
**答：** 我们测试了像“⠠de╱Aura”这样的符号。虽然它们看起来很独特，但其他人很难将它们复制粘贴或手动输入到搜索引擎中。 `SL5net Aura` 是最强大的“人与搜索引擎”桥梁。

### 问：Emoji（🗣）够专业吗？
**答：** 在 95% 的现代开发环境（GitHub、Discord、Slack）中，表情符号是标准的。如果您处于高合规性企业环境（例如银行业），我们建议使用更简洁的版本：
`[ SL5net 光环 ]`
