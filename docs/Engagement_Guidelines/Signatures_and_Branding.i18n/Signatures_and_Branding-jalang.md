# コミュニケーション ガイド: 署名とブランディング

**Aura** の認知度を押し付けがましくなく広めるために、チャット署名を通じて「パッシブ ブランディング」戦略を採用しています。これにより、ユーザーは会話の内容に集中したままプロジェクトを簡単に見つけることができます。

## 1. オーラの署名
チャットで翻訳または自動化機能を使用する場合、Aura はメッセージに小さな署名を追加できます。

**推奨される形式:**
> `🗣SL5net ⟫ オーラ`

### 記号の内訳:
* **`🗣` (Speaking Head):** このメッセージが処理または翻訳されたことを示す視覚的なインジケーター。それは「コミュニケーション/言語」を信号します。
* **`SL5net`:** 固有の名前空間。これは検索可能性にとって非常に重要です。
* **`⟫` (二重右括弧):** CLI/シェル オペレーターのように見える技術的な「パイプ」シンボル。これが技術的なツールであることを強調します。
* **`Aura`:** プロジェクト名。

---

## 2. なぜリンクではなく「SL5net Aura」なのでしょうか？
多くのチャット プラットフォーム (Matrix、Discord、Telegram など) には、URL またはドメインのような構造に対する厳密なフィルターがあります。
* **検索性:** Google または GitHub で「SL5net Aura」を検索すると、100% の成功率でリポジトリが見つかります。
* **フィルターセーフ:** 「.com」リンクの代わりに一意の検索語を使用することで、自動モデレーターによって「スパム」または「広告ボット」としてフラグが立てられることを回避します。
* **低摩擦:** 読みやすく、検索バーに簡単に入力できます。

## 3. 署名エチケット (FAQ)
チャットパートナーに迷惑をかけずに署名を効果的に使用する方法:

### Q: すべてのメッセージに署名が必要ですか?
**A:** いいえ。ペースの速いチャットでは、一定の署名がスパムとして認識される可能性があります。
* **ベスト プラクティス:** 新しい会話の最初のメッセージ、または特定の「価値の高い」翻訳済みメッセージに対してのみ署名を有効にします。
* **設定:** プライベートまたは非常にフォーマルなビジネス会議の場合は、設定内の「#signatur」トグルを使用してオフにします。

### Q: 点字や複雑な Unicode 文字を使用しないのはなぜですか?
**A:** `⠠de╱Aura` のようなシンボルをテストしました。これらはユニークに見えますが、他の人がコピー＆ペーストしたり、検索エンジンに手動で入力したりするのは困難です。 「SL5net Aura」は、最も堅牢な「人間から検索エンジンへ」のブリッジです。

### Q: 絵文字 (🗣) は十分にプロフェッショナルですか?
**A:** 最新の開発環境 (GitHub、Discord、Slack) の 95% では、絵文字が標準です。コンプライアンスの高い企業環境 (銀行など) にいる場合は、よりクリーンなバージョンをお勧めします。
『SL5netオーラ』
