# البدء مع SL5 Aura

                                                      ## ما هي SL5 Aura؟

SL5 Aura هو مساعد صوتي أول بدون اتصال يقوم بتحويل الكلام إلى نص (STT) ويطبق قواعد قابلة للتكوين لتنظيف الإخراج وتصحيحه وتحويله.

إنه يعمل بدون واجهة المستخدم الرسومية – كل شيء يعمل عبر واجهة سطر الأوامر (CLI) أو وحدة التحكم.

                                                           ## كيف يعمل

```
Microphone → Vosk (STT) → Maps (Pre) → LanguageTool → Maps (Post) → Output
```

                           1. **Vosk** يحول كلامك إلى نص خام
2. **الخرائط المسبقة** تنظف النص وتصححه قبل التدقيق الإملائي
3. **أداة اللغة** تعمل على إصلاح القواعد النحوية والإملائية
4. **الخرائط اللاحقة** تطبق التحولات النهائية
5. **الإخراج** هو النص النظيف النهائي (وتحويل النص إلى كلام بشكل اختياري)

                                                 ## خطواتك الأولى

                                                 ### 1. ابدأ الهالة
```bash
python main.py
```

                    ### 2. اختبار مع إدخال وحدة التحكم
                      اكتب `s` متبوعًا بالنص الخاص بك:
```
s hello world
```

                         ### 3. شاهد القاعدة أثناء العمل
افتح `config/maps/koans_deutsch/01_koan_erste_schritte/de-DE/FUZZY_MAP_pre.py`

قم بإلغاء تعليق القاعدة الموجودة بالداخل واختبرها مرة أخرى. ماذا يحدث؟

                                                     ## فهم القواعد

القواعد موجودة في "config/maps/" في ملفات Python التي تسمى "FUZZY_MAP_pre.py" أو "FUZZY_MAP.py".

                                      القاعدة تبدو كالتالي:
```python
('Hello World', r'\bhello world\b', 0, {'flags': re.IGNORECASE})
#   ^output        ^pattern          ^threshold  ^case-insensitive
```

يأتي **الإخراج** أولاً – وسترى على الفور ما تنتجه القاعدة.

تتم معالجة القواعد ** من أعلى إلى أسفل **. أول مباراة كاملة (`^...$`) توقف كل شيء.

                                 ## Koans – التعلم بالممارسة

Koans عبارة عن تمارين صغيرة في `config/maps/koans_deutsch/` و`config/maps/koans_english/`.

                              كل كوان يعلم مفهوما واحدا:

                                                | كوان | الموضوع |
                                                                    |---|---|
| 01_koan_erste_schritte | القاعدة الأولى، مباراة كاملة، توقف خط الأنابيب |
                    | 02_koan_listen | قوائم، قواعد متعددة |
| 03_koan_schwierige_namen | الأسماء الصعبة، المطابقة الصوتية |

                              ابدأ بـ Koan 01 ثم واصل طريقك.

- يتم تشغيل القواعد في `FUZZY_MAP_pre.py` قبل ** التدقيق الإملائي - وهي مفيدة لإصلاح أخطاء STT
- يتم تشغيل القواعد الموجودة في `FUZZY_MAP.py` ** بعد ** التدقيق الإملائي - وهي جيدة للتنسيق
