# Suche nach Befehlen
Sie können alle möglichen Sprachbefehle leicht finden.
Sagen Sie: „Aura Suche“.
Es öffnet sich ein Terminalfenster mit allen Regeln.
Geben Sie ein Schlüsselwort wie „LibreOffice“ oder „Git“ ein.
Sie sehen alle Befehle für dieses Thema.
Drücken Sie die Eingabetaste, um die Regeldatei zu bearbeiten.
