# Procurando por Comandos
Você pode encontrar facilmente todos os comandos de voz possíveis.
Diga: "Aura Suche".
Uma janela de terminal é aberta com todas as regras.
Digite uma palavra-chave como “LibreOffice” ou “Git”.
Você verá todos os comandos para esse tópico.
Pressione Enter para editar o arquivo de regras.
