# Wyszukiwanie poleceń
Z łatwością znajdziesz wszystkie możliwe polecenia głosowe.
Powiedz: „Aura Suche”.
Otworzy się okno terminala ze wszystkimi regułami.
Wpisz słowo kluczowe, np. „LibreOffice” lub „Git”.
Zobaczysz wszystkie polecenia dla tego tematu.
Naciśnij klawisz Enter, aby edytować plik reguły.
