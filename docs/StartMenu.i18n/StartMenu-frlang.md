# Recherche de commandes
Vous pouvez trouver facilement toutes les commandes vocales possibles.
Dites : « Aura Suche ».
Une fenêtre de terminal s'ouvre avec toutes les règles.
Tapez un mot-clé comme « LibreOffice » ou « Git ».
Vous verrez toutes les commandes pour ce sujet.
Appuyez sur Entrée pour modifier le fichier de règles.
