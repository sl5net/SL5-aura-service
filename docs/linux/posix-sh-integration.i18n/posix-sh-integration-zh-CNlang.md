## POSIX / Dash 特定注释

- 为了最大程度的兼容性，此处**不**使用“local”。所有变量仅按照约定在函数范围内；从技术上讲，它们在严格的 POSIX sh 中是全局的。
- 将参数传递给命令时，`$@` 优于 `$*`，以保留带引号参数的正确单词分割。
- 执行 Kiwix 帮助程序脚本时，`bash` 被替换为 `sh`，以保留在 POSIX 工具链中。
- 此配置文件最好放置在“~/.profile”中，该文件由登录 shell 获取。对于交互式非登录 shell，您的发行版可能使用 `~/.shrc` — 检查您的系统文档。

＃＃ 特征

- **动态路径**：通过“/tmp”标记文件自动查找项目根目录。
- **自动重启**：如果后端关闭，它会尝试运行“start_service”和本地维基百科服务。
- **智能超时**：首先尝试 2 秒快速响应，然后退回到 70 秒深度处理模式。
