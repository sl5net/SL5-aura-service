## मछली-विशिष्ट नोट्स

- मछली वेरिएबल असाइनमेंट के लिए `VAR=वैल्यू` के बजाय `सेट VAR वैल्यू` का उपयोग करती है।
- शर्तें `[ ]` और `fi` के बजाय `test` और `end` ब्लॉक का उपयोग करती हैं।
- तर्क पारित करने के लिए `$argv` `$*` / `$@` को प्रतिस्थापित करता है।
- निकास कोड के लिए `$status` `$?` का स्थान लेता है।
- `या` / `और` को सशर्त अभिव्यक्तियों में `||` / `&&` से बदलें।
- मछली `स्थानीय` का उपयोग **नहीं** करती है - फ़ंक्शन के अंदर सभी चर डिफ़ॉल्ट रूप से स्थानीय होते हैं।
