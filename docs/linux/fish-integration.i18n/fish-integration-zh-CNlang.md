# 鱼壳整合

为了更轻松地与 STT（语音转文本）CLI 进行交互，您可以向 Fish 配置添加快捷方式功能。这使您只需在终端中输入“您的问题”即可。

## 设置说明

Fish shell 将函数存储为单独的文件。推荐的方法是创建专用的函数文件。

1、创建函数文件（目录不存在会自动创建）：
__代码_块_0__

2. 将以下块粘贴到文件中：

__代码_块_1__

3. 该功能在所有新的 Fish 会话中立即可用。要在当前会话中加载它而不打开新终端：
__代码_块_2__

## 鱼类特定注释

- Fish 使用“set VAR value”而不是“VAR=value”进行变量赋值。
- 条件使用“test”和“end”块而不是“[]”和“fi”。
- `$argv` 替换 `$*` / `$@` 用于参数传递。
- `$status` 替换了退出代码的 `$?`。
- 条件表达式中的“or”/“and”替换“||”/“&&”。
- Fish **不**使用“local”——默认情况下，函数内的所有变量都是本地的。

＃＃ 特征

- **动态路径**：通过“/tmp”标记文件自动查找项目根目录。
- **自动重启**：如果后端关闭，它会尝试运行“start_service”和本地维基百科服务。
- **智能超时**：首先尝试 2 秒快速响应，然后退回到 70 秒深度处理模式。
