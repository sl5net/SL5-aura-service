                     ## 3. تقرير الخطأ: مشكلات التبعية
                          ### المشكلة: فشل تثبيت "libtorrent".
في بيئة **Manjaro/Arch Linux** الحالية، لا يمكن تثبيت محرك BitTorrent ("libtorrent") عبر مديري الحزم القياسيين.

                                     * **الأوامر المُحاولة:**
* `sudo pacman -S python-libtorrent` -> `لم يتم العثور على الهدف`
* `pamac build python-libtorrent-rasterbar` -> `لم يتم العثور على الهدف`
* `pamac build python-libtorrent` -> `لم يتم العثور على الهدف`
* **السبب الجذري:** غالبًا ما تتم صيانة روابط Python لـ "libtorrent" في الأنظمة المستندة إلى Arch بشكل سيئ في عمليات إعادة الشراء الرسمية أو تتطلب أدوات مساعدة/أدوات إنشاء AUR محددة (\'base-devel`) مفقودة حاليًا أو تمت تهيئتها بشكل خاطئ.
* **التأثير:** ميزات BitTorrent (P2P وWeb-Seeds) غير نشطة حاليًا. يظل البرنامج النصي يعمل بكامل طاقته عبر **HTTP الاحتياطي**.
