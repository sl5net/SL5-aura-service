### 2. `टिल्डे_शेल_हैक.एमडी`
*यह फ़ाइल `टिल्डे/` प्रतिस्थापन और Zsh एकीकरण विजेट दोनों का दस्तावेजीकरण करते हुए, प्रमुख विशेष वर्णों पर ऑरा के प्रतिबंध के समाधान की व्याख्या करती है।*

__CODE_ब्लॉक_0__
# ==========================================================
# एंटर पर स्वचालित "टिल्डे/" -> "~/" प्रतिस्थापन
# ==========================================================
स्वीकार-लाइन-ओवरराइड() {
बफ़र='${बफ़र//टिल्डे\/~/}'
ज़ले .स्वीकार-पंक्ति
}
