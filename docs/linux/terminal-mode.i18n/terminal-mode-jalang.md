# ターミナルモード (言語除外)

ターミナル モードは、特定の言語パックがインストールされていない、または音声/テキスト処理ユニット用に構成されていない構成状態です。

## 有効にする方法
初期セットアップまたは言語選択スクリプト中に、**主言語**の入力を求められたら、次のように入力します。
-「ん」
- 「なし」
- `0`

## 効果
- **EXCLUDE_LANGUAGES** は「all」に設定されます。
- 言語固有のモデル (Whisper モデルや Vosk モデルなど) はダウンロードまたは初期化されません。
- システムは「ターミナルのみ」モードで動作します。これは、ローディスク環境や、ローカライズされた音声サポートなしでコア CLI ツールのみが必要な場合に便利です。
