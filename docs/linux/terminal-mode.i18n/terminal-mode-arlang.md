# الوضع الطرفي (استبعاد اللغة)

الوضع الطرفي هو حالة تكوين حيث لا يتم تثبيت أو تكوين أي حزم لغات محددة لوحدات معالجة الكلام/النص.

                                                 ## كيفية التمكين
أثناء الإعداد الأولي أو البرنامج النصي لاختيار اللغة، عند المطالبة بـ **اللغة الأساسية**، أدخل:
                                                                       - `ن`
                                                             - "لا شيء".
                                                                        - `0`

                                                        ## التأثيرات
                   - **EXCLUDE_LANGUAGES** تم ضبطها على `الكل`.
- لن يتم تنزيل أو تهيئ أي نماذج خاصة بلغة معينة (مثل نماذج Whisper أو Vosk).
- يعمل النظام في وضع "المحطة الطرفية فقط"، وهو مفيد لبيئات الأقراص المنخفضة أو عندما تكون أدوات CLI الأساسية فقط مطلوبة دون دعم الكلام المترجم.
