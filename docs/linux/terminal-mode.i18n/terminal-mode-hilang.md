# टर्मिनल मोड (भाषा बहिष्करण)

टर्मिनल मोड एक कॉन्फ़िगरेशन स्थिति है जहां वाक्/पाठ प्रसंस्करण इकाइयों के लिए कोई विशिष्ट भाषा पैक स्थापित या कॉन्फ़िगर नहीं किया जाता है।

## कैसे सक्षम करें
प्रारंभिक सेटअप या भाषा चयन स्क्रिप्ट के दौरान, जब **प्राथमिक भाषा** के लिए कहा जाए, तो दर्ज करें:
- `एन`
- `कोई नहीं`
- `0`

## प्रभाव
- **EXCLUDE_LANGUAGES** `सभी` पर सेट है।
- कोई भी भाषा-विशिष्ट मॉडल (जैसे व्हिस्पर या वोस्क मॉडल) डाउनलोड या प्रारंभ नहीं किया जाएगा।
- सिस्टम "टर्मिनल ओनली" मोड में काम करता है, जो लो-डिस्क वातावरण के लिए उपयोगी है या जब स्थानीयकृत वाक् समर्थन के बिना केवल कोर सीएलआई टूल की आवश्यकता होती है।
