# 터미널 모드(언어 제외)

터미널 모드는 음성/텍스트 처리 장치에 대해 특정 언어 팩이 설치되거나 구성되지 않은 구성 상태입니다.

## 활성화 방법
초기 설정 또는 언어 선택 스크립트 중에 **기본 언어**를 묻는 메시지가 나타나면 다음을 입력합니다.
-`n`
- '없음'
- `0`

## 효과
- **EXCLUDE_LANGUAGES**는 '모두'로 설정됩니다.
- 언어별 모델(예: Whisper 또는 Vosk 모델)은 다운로드되거나 초기화되지 않습니다.
- 시스템은 "터미널 전용" 모드에서 작동하며, 디스크가 부족한 환경이나 지역화된 음성 지원 없이 핵심 CLI 도구만 필요한 경우에 유용합니다.
