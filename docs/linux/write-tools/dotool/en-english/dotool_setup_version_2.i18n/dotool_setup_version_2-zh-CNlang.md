## 故障排除和注释
- **缺少字符：** 如果跳过特殊字符（如变音符号），请将 `dotool_typedelay` 增加到 5 或 10。
- **应用程序兼容性：** 某些应用程序（电子、浏览器）可能需要更高的延迟才能正确注册快速输入。
- **Wayland 支持：** `dotool` 是 Wayland 所需的后端，因为 `xdotool` 不支持它。
- **自动回退：** 如果未正确安装或配置“dotool”，脚本会自动回退到“xdotool”。
