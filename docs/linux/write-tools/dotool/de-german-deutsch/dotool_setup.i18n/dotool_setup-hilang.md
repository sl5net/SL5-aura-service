## ist dotool था?

यह `xdotool` और x11 और वेलैंड के बीच का फ़ंक्शन है।

##हिनवाइज़

- **उम्लाउते और सोंडरज़ेइचेन:** `प्रकार विलंब 2` (टूल-डिफ़ॉल्ट करें) यह प्रभावी है।
`टाइपडेले 0` को देखते हुए ä, ö, ü, ß वर्लोरेन गेहेन।
- **ज़िलानवेन्दुंग के लिए क्या आवश्यक है?** मांचे ऐप्स (जेड. बी. इलेक्ट्रॉन, ब्राउज़र-इनपुट्स)
विलंबित समय से पहले. इस फ़ॉल में `dotool_typedelay = 5` या अन्य लाभ।
- **वेलैंड:** वेलैंड पर डॉट टूल का उपयोग करें, xdotool अभी नहीं।
- **फ़ॉलबैक:** जब कोई टूल इंस्टॉल नहीं होता है, तो स्क्रिप्ट स्वचालित रूप से `xdotool` से जुड़ जाती है।
