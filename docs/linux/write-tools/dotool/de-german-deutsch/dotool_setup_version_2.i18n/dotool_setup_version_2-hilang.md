## ist dotool था?
`डॉटूल` एक कार्य है जो टेस्टाट्यूरिंगबेन से सिमुलेशन के लिए उपयुक्त है। मैं `xdotool` को सीधे `uinput` के माध्यम से कर्नेल से जोड़ता हूं और **X11 और वेलैंड** तक सार्वभौमिक रूप से कार्य करता हूं।
