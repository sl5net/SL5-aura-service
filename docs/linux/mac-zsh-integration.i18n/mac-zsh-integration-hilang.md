# macOS Zsh शैल एकीकरण

> **macOS Catalina (10.15) के बाद से डिफ़ॉल्ट शेल।** यदि आप macOS Mojave या इससे पहले के संस्करण पर हैं, तो इसके बजाय [macOS Bash Integration](.././mac-bash-integration.i18n/mac-bash-integration-hilang.md) गाइड देखें।

एसटीटी (स्पीच-टू-टेक्स्ट) सीएलआई के साथ इंटरैक्ट करना आसान बनाने के लिए, आप अपने `~/.zshrc` में एक शॉर्टकट फ़ंक्शन जोड़ सकते हैं। यह आपको टर्मिनल में बस "आपका प्रश्न" टाइप करने की अनुमति देता है।

- **`pgrep`** डिफ़ॉल्ट रूप से macOS पर उपलब्ध है।

- **पायथन पथ**: सुनिश्चित करें कि आपका वर्चुअल वातावरण `$PROJECT_ROOT/.venv` पर सेट है। यदि आप Python को `pyenv` या `conda` के साथ प्रबंधित करते हैं, तो `PY_EXEC` को तदनुसार समायोजित करें।

## विशेषताएँ

- **डायनामिक पथ**: स्वचालित रूप से `/tmp` मार्कर फ़ाइल के माध्यम से प्रोजेक्ट रूट ढूंढता है।
- **ऑटो-रीस्टार्ट**: यदि बैकएंड डाउन है, तो यह `start_service` और स्थानीय विकिपीडिया सेवाओं को चलाने का प्रयास करता है।
- **स्मार्ट टाइमआउट्स**: पहले 2-सेकंड की त्वरित प्रतिक्रिया का प्रयास करता है, फिर 70-सेकंड के डीप प्रोसेसिंग मोड पर वापस आ जाता है।
