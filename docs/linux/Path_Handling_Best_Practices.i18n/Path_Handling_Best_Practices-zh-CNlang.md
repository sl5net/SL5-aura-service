# 主目录和跨平台路径处理

Aura 旨在跨多个操作系统运行。为了确保无论您运行的是 Linux、macOS 还是 Windows，文件系统导航命令都能正常工作，路径字符串在注册到活动模糊映射之前会被动态解析。

---

## 路径标准化逻辑 (`FUZZY_MAP_pre.py`)

动态路径映射逻辑依赖于以下标准实践：

### 1.波浪号减少（POSIX）
在 POSIX 兼容系统（Linux 和 macOS）上，与用户主目录匹配的绝对路径（例如“/home/username/”）在启动时会转换为“~”相对路径。这使得字符串长度更短，并使生成的规则可以在同一操作系统上的不同用户之间移植：

### 2. 绝对路径保留 (Windows)
Windows 无法可靠地评估标准命令提示符 (`cmd.exe`) 或 PowerShell 环境中的 `~` 字符。因此，当插件检测到 Windows 环境（`sys.platform == 'win32'`）时，它会保留完全限定的绝对路径（例如，`C:\Users\username\...`）以确保命令执行不会失败。

### 3. 正斜杠标准化 (`as_posix()`)
Aura 在配置映射内部使用 POSIX 风格的正斜杠 (`/`)。该脚本利用 Python 的“pathlib.Path.as_posix()”方法标准化所有依赖于操作系统的路径分隔符，该方法会自动清理 Windows 环境中的反斜杠 (“\”)。
