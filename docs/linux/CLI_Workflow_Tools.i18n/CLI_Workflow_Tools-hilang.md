# सीएलआई वर्कफ़्लो टूल्स इंस्टालेशन गाइड

पथ नेविगेटर प्लगइन में कुछ क्रियाएं अस्पष्ट खोज करने, फ़ाइलों को सूचीबद्ध करने और क्लिपबोर्ड में हेरफेर करने के लिए बाहरी कमांड-लाइन उपयोगिताओं पर निर्भर करती हैं। यदि ये उपकरण गायब हैं, तो आपको सिस्टम कंसोल में एक चेतावनी दिखाई देगी।

नीचे प्रत्येक समर्थित ऑपरेटिंग सिस्टम के लिए इंस्टॉलेशन निर्देश दिए गए हैं।

## आवश्यक उपयोगिताएँ

* **`fzf`**: सामान्य प्रयोजन कमांड-लाइन फ़ज़ी फ़ाइंडर।
* **`खोजें`** (या `एफडी`): मानक फ़ाइल-खोज उपयोगिता।
* **क्लिपबोर्ड टूल**: आउटपुट को सीधे आपके सिस्टम क्लिपबोर्ड पर पाइप करने के लिए उपयोग किया जाता है।
* **लिनक्स:** `xclip` (X11 वातावरण की आवश्यकता है)।
* **macOS:** `pbcopy` (पूर्व-स्थापित)।
* **विंडोज़:** `क्लिप` (पूर्व-स्थापित)।
* **`फ़ाइल`**: पूर्ण टर्मिनल पूर्वावलोकन के लिए फ़ाइल प्रकार निर्धारित करता है।
