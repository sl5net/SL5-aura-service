# यूनिफाइड_सिंक
## सेटिंग्स.AUDIO_INPUT_DEVICE == 'MIC_AND_DESKTOP
### विंडोज़ के लिए रणनीति बनाएं (समाधान)
विंडोज़ में **ओबीएस-मॉनिटरिंग** के साथ संयोजन में **वर्चुअल ऑडियो केबल** शामिल है।

**दास सेटअप:**
1. **वर्चुअल केबल इंस्टालेशन:** (z. B. *VB-केबल*)। विंडोज़ पर "यूनिफाइड_सिंक" का उपयोग किया जाता है।
2. **ओबीएस मॉनिटरिंग:** जब ओबीएस स्थापित किया गया था *ऑडियो -> मॉनिटरिंग -> मॉनिटरिंग-गेराट* को "वर्चुअल केबल" कहा जाता था।
3. **मिक्स एरस्टेलन:** फर जेडे क्वेले इन ओबीएस (माइक, डेस्कटॉप) स्टेलस्ट डू इन डेन *एर्वेइटरटेन ऑडियोइगेन्सचाफ्टन* „मॉनिटरिंग अंड ऑस्गेबे” ईन।
4. **पायथन:** अपनी सेटिंग्स में `AUDIO_INPUT_DEVICE = "CABLE आउटपुट"` सेट करें।

### विश्लेषण करें
* **परिवर्तन:** ओबीएस पूर्ण मिश्रण। विंडोज़ नोट्स के लिए अधिक जटिल पायथन-एंडराउंड।
* **नचटेइल:** डेर बेनुत्ज़र मुस्स ओबीएस इम हिंटरग्रंड लॉफेन लासेन।

विंडोज-यूजर के लिए स्थिरता की आवश्यकता है।

**विंडोज (पॉवरशेल) के लिए केबल नाम बदलें:**
`ऑडियोडिवाइस-सूची प्राप्त करें`
*(हिनवीस: ऑडियोडिवाइससीएमडीलेट्स-मॉड्यूल द्वारा समर्थित)।*
