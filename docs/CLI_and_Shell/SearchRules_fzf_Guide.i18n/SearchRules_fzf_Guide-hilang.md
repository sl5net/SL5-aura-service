## fzf टेस्टेनकुर्ज़ेल

| स्वाद | एक्शन |
|---|---|
| टिप्पन | सुचानफ्रेज ईन्गेबेन |
| `दर्ज करें` | डेटी इम संपादक ओफ़नेन (केट) |
| `Ctrl+G` | ब्राउज़र में GitHub डाउनलोड करें |
| `Ctrl+Z` | वोरहेरिज सुचानफ्रेज ऑस हिस्ट्री |
| `Ctrl+Y` | सबसे पहले सुचानफ्रेज का इतिहास |
| `Ctrl+P` | वोरहेरिगर हिस्ट्री-एंट्रैग |
| `Ctrl+N` | नैचस्टर हिस्ट्री-एंट्रैग |
| `Ctrl+A` | एले ट्रेफ़र ऑस्वाहलेन |
| `Ctrl+C` | अब्ब्रेचेन |
| `Ctrl+←/→` | वॉर्टवाइज़ नेविगेरेन |
| `Ctrl+बैकस्पेस` | वोर्ट लोस्चेन (लिंक) |
| `Ctrl+डिलीट` | वोर्ट लोस्चेन (रेच्ट्स) |

## पूर्वावलोकन-फ़ेंस्टर

दास फ़ेज़फ़-फ़ेंस्टर ज़िगट ओबेन (50%) एक वोर्सचाऊ डेस ट्रेफ़र्स:
- 5 ज़िलेन वोर अंड नच डेम मैच
- ट्रेफ़र-ज़ीले का उद्देश्य `>` चिह्न है
- ज़िलेन्नुमर्न वेर्डन एंजेज़िगट

## सुचवेर्लॉफ़

स्वचालित रूप से शुरू करने के लिए सुचानफ्रेज का उपयोग शुरू करने से पहले।
डेर वेरलॉफ़ `~/.search_rules_history` में शामिल हैं।
