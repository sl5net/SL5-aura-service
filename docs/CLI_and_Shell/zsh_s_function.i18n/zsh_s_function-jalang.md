# Zsh 関数: s() - 適応タイムアウトを備えた KI クライアント

英語 (英語)
目的

この Zsh 関数は、Python クライアント (cli_client.py) のラッパーとして機能し、堅牢なエラー処理と適応型タイムアウト戦略を実装します。サービス接続エラーを迅速に検出し、完全な AI 応答 (最大 70 秒) が確実にキャプチャされるように設計されています。
キーロジック

この関数は、堅牢性を確保するために次の 2 つのシェル機能に依存しています。

タイムアウト: スクリプトが無期限にハングするのを防ぎ、迅速なエラー検出を可能にします。

mktemp / 一時ファイル: 終了後にスクリプトの出力をファイルから読み取ることで、シェルの出力バッファリングの問題を回避します。

使用法
コードバッシュ

  
s <質問のテキスト>
# 例: s コンピュータ Guten Morgen
