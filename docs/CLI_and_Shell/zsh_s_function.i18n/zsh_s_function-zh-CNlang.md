# Zsh 功能：s() - KI-Client mit Adaptivem 超时

英语（英语）
目的

此 Zsh 函数充当 Python 客户端 (cli_client.py) 的包装器，并实现强大的错误处理和自适应超时策略。它旨在快速检测服务连接错误并确保捕获完整的 AI 响应（最多 70 秒）。
关键逻辑

该函数依赖于两个 shell 功能来实现稳健性：

超时：防止脚本无限期挂起并允许快速错误检测。

mktemp / 临时文件：通过在终止后从文件中读取脚本的输出来绕过 shell 输出缓冲问题。
