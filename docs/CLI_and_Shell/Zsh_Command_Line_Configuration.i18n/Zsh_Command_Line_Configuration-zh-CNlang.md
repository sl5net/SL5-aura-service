### 3. 三个命令的使用

|命令|功能|安全等级|示例|
| :--- | :--- | :--- | :--- |
| **`sl`** | **标准输出：** 执行服务并将整个输出直接打印到控制台。 | **安全** | `sl 什么是房子`（打印：“房子是......”）|
| **`slz`** | **安全执行准备：** 执行服务并将输出（例如 shell 命令）粘贴到 Zsh 输入行中，以供查看或执行。 | **安全/准备** | `slz git` （粘贴：`git add . && git commit...` **但不运行它**。） |
| **`slxXsoidfuasdzof`** | **立即执行：** 执行服务并立即将输出作为 shell 命令运行。使用神秘名称作为安全措施。 | **危险** | `slxXsoidfuasdzof git` （立即运行 `git add...` 命令。） |
