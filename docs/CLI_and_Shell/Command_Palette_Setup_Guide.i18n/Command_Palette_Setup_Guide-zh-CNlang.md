# 命令面板和地图搜索指南

本指南介绍了如何为 SL5 Aura 设置和使用系统范围内、与位置无关的**命令面板**。它允许您以交互方式搜索映射规则，从本地 SQLite 缓存查看实时执行预览，并立即在活动光标处键入选定的输出。

## 先决条件

确保以下后台服务和工具已安装并处于活动状态：
1. **`fzf`**（模糊查找器）
2. **CopyQ**（剪贴板管理器，用于全局热键编排）
3. **`type_watcher.sh`**（Aura后台打字守护进程）

---

## CopyQ 全局快捷方式设置

要从任何活动窗口（例如浏览器或文本编辑器）立即启动命令面板，请在 CopyQ 中配置全局热键：

1. 打开 **CopyQ** 并按 `F6`（或转到 **命令** / **Befehle**）。
2. 单击 **添加** (Hinzufügen) 并将其命名为“Aura Command Palette”。
3. 设置您所需的**全局快捷键**（例如“Meta+S”或“Ctrl+Alt+S”）。
