# لوحة الأوامر ودليل البحث عن الخريطة

يشرح هذا الدليل كيفية إعداد واستخدام **Command Palette** على مستوى النظام والمستقل عن الموقع لـ SL5 Aura. فهو يتيح لك البحث في قواعد الخريطة بشكل تفاعلي، ورؤية معاينات التنفيذ المباشر من ذاكرة التخزين المؤقت SQLite المحلية، وكتابة الإخراج المحدد على الفور في المؤشر النشط.

                                       ## المتطلبات الأساسية

تأكد من تثبيت الخدمات والأدوات الأساسية التالية وتنشيطها:
                                     1. **`fzf`** (الباحث الغامض)
2. **CopyQ** (مدير الحافظة، يستخدم لتنسيق مفاتيح التشغيل السريع العالمية)
     3. **`type_watcher.sh`** (برنامج الكتابة بخلفية Aura)
