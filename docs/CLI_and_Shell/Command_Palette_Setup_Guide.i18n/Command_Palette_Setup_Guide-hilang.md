# कमांड पैलेट और मैप सर्च गाइड

यह मार्गदर्शिका बताती है कि SL5 ऑरा के लिए सिस्टम-व्यापी, स्थान-स्वतंत्र **कमांड पैलेट** को कैसे सेट अप और उपयोग करें। यह आपको अपने मानचित्र नियमों को अंतःक्रियात्मक रूप से खोजने, स्थानीय SQLite कैश से लाइव निष्पादन पूर्वावलोकन देखने और तुरंत अपने सक्रिय कर्सर पर चयनित आउटपुट टाइप करने की अनुमति देता है।

## पूर्वावश्यकताएँ

सुनिश्चित करें कि निम्नलिखित पृष्ठभूमि सेवाएँ और उपकरण स्थापित और सक्रिय हैं:
1. **`fzf`** (फजी फाइंडर)
2. **कॉपीक्यू** (क्लिपबोर्ड प्रबंधक, वैश्विक हॉटकी ऑर्केस्ट्रेशन के लिए उपयोग किया जाता है)
3. **`type_watcher.sh`** (ऑरा बैकग्राउंड टाइपिंग डेमॉन)

## CopyQ ग्लोबल शॉर्टकट सेटअप

किसी भी सक्रिय विंडो (उदाहरण के लिए, आपका ब्राउज़र या टेक्स्ट एडिटर) से तुरंत कमांड पैलेट लॉन्च करने के लिए, CopyQ में एक वैश्विक हॉटकी कॉन्फ़िगर करें:

1. **CopyQ** खोलें और `F6` दबाएँ (या **कमांड** / **बेफ़हले** पर जाएँ)।
2. **जोड़ें** (हिंज़ुफुगेन) पर क्लिक करें और इसे `ऑरा कमांड पैलेट` नाम दें।
3. अपना वांछित **वैश्विक शॉर्टकट** सेट करें (जैसे, `मेटा+S` या `Ctrl+Alt+S`)।
4. **प्रकार** को `कमांड` (बेफ़ल) पर सेट करें।
