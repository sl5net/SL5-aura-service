# 명령어 팔레트 & 지도 검색 가이드

이 가이드에서는 SL5 Aura용 시스템 전체의 위치 독립적인 **명령 팔레트**를 설정하고 사용하는 방법을 설명합니다. 이를 통해 맵 규칙을 대화형으로 검색하고, 로컬 SQLite 캐시에서 실시간 실행 미리보기를 확인하고, 활성 커서에 선택한 출력을 즉시 입력할 수 있습니다.

다음 백그라운드 서비스 및 도구가 설치되어 활성화되어 있는지 확인하세요.
1. **`fzf`** (퍼지 파인더)

## CopyQ 글로벌 단축키 설정

활성 창(예: 브라우저 또는 텍스트 편집기)에서 명령 팔레트를 즉시 실행하려면 CopyQ에서 전역 단축키를 구성하세요.

1. **CopyQ**를 열고 `F6`을 누르세요(또는 **Commands** / **Befehle**로 이동).
2. **추가**(Hinzufügen)를 클릭하고 이름을 `Aura Command Palette`로 지정합니다.
3. 원하는 **전역 바로가기**(예: `Meta+S` 또는 `Ctrl+Alt+S`)를 설정합니다.
4. **유형**을 `명령`(Befehl)으로 설정합니다.
5. 명령 상자에 다음 JavaScript 코드를 붙여넣습니다.
