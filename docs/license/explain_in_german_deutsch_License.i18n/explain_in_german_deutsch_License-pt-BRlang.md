# Die License erklärt in deutscher Sprache

A **Licença Apache, Versão 2.0** é definitivamente **Software Livre** (Software Gratuito) e também **Software de Código Aberto**.

Aqui estão os detalhes, warum das so ist und was das für dich bedeutet:

### 1. Oficial de Anerkennung
* **FSF (Free Software Foundation):** Use o Apache 2.0 como licenciado para “Software Gratuito”. Você está ciente de **não compatível com a GPL Versão 2**, mas compatível com a **GPL Versão 3**.
* **OSI (Open Source Initiative):** Use o Apache 2.0 como uma licença de código aberto popular e padronizada.

### 2. Darf man damit machen? (Os Freiheiten)
Unter der Apache 2.0 Lizenz darf jeder (auch Firmen):
* **O software para todos os usuários:** Comercial, privado, estagiário – todos criados.
* **O código é alterado:** Você pode usar a ferramenta como quiser.
* **O software é copiado e informado:** Você deve saber mais.
* **Sogar verkaufen:** Seu primeiro software, baseado no código Apache, é usado.

### 3. Die Besonderheit: "Permissivo" (Freizügig)
Im Gegensatz zur GPL (General Public License) que tem o Apache-Lizenz **keinen "Copyleft"-Effekt**.
* É necessário: Se o Aura (até o Apache 2.0) for nimmst und veränderst, **deve** seu novo código não deve ser usado como código-fonte aberto. Você também pode obter um produto comercializável.

### 4. Ein großer Vorteil: Die Patentklausel
A licença Apache 2.0 é uma crença firme (frequentemente crente no MIT ou BSD), e ela possui uma **ausdrückliche Patenterteilung** enthält.
* Embora o código do projeto seja aprovado, ele gera automaticamente uma licença para todas as patentes, e esse código é interrompido.
* Das schützt die Nutzer vor "Patent-Trollen". Se alguém disser que a patente foi aprovada, o próprio projeto foi aprovado, ele é automaticamente o seu direito e o software.

### Fassammenfassung:
No **Apache 2.0**, o sinal mais profissional é:

1. **Rechtssicherheit:** Besonders für große Firmen (como Automotivo ou Financeiro) são as "sicherste" Lizenz.
2. **Flexibilidade:** Se você pode ter problemas com ferramentas em seus conjuntos de ferramentas internos integrados, sua preocupação com o efeito copyleft deve ser necessária.

**Fazit:** É um software livre e de código aberto (Free and Open Source Software) im besten Sinne. É "frei" sowohl im Sinne von "kostenlos" como também im Sinne von "Freiheit".
