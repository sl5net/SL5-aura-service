# Die License erklärt en alemán Sprache

La **Licencia Apache, Versión 2.0** es **Software libre** definitivo y también **Software de código abierto**.

Aquí sind die Detalles, warum das so ist und was das für dich bedeutet:

### 1. Anerkennung oficial
* **FSF (Free Software Foundation):** Incluye Apache 2.0 como Lizenz para "software gratuito". Esto es **no compatible con la versión 2 de GPL**, pero no compatible con la versión 3 de GPL**.
* **OSI (Iniciativa de código abierto):** Esta versión de Apache 2.0 es una de las más populares y estandarizadas de código abierto.

### 2. ¿Era darf man damit machen? (Die Freiheiten)
Bajo el Apache 2.0 Lizenz darf jeder (también Firmen):
* **Die Software für jeden Zweck nutzen:** Kommerziell, privat, intern – alles erlaubt.
* **Den Code verändern:** Du kannst das Tool anpassen, wie du willst.
* **Die Software kopieren and verbreiten:** Du darfst sie weitergeben.
* **Sogar verkaufen:** Du darfst Software, die auf Apache-Code basiert, verkaufen.

### 3. Die Besonderheit: "Permisivo" (Freizügig)
Estoy Gegensatz zur GPL (Licencia Pública General) que el Apache-Lizenz **no tiene efecto "Copyleft"**.
* Das bedeutet: Wenn du Aura (bajo Apache 2.0) nimmst und veränderst, **musst** du deinen neuen Code nicht zwingend wieder como Open Source veröffentlichen. Du kannst ihn auch in ein geschlossenes, kommerzielles Produkt einbauen.

### 4. Un gran voto: Die Patentklausel
Apache 2.0 Lizenz ist bei Firmen sehr beliebt (a menudo beliebter como MIT o BSD), weil sie una **ausdrückliche Patenterteilung** enthält.
* Sin embargo, der Code zum Projekt beiträgt, gewährt den Nutzern automatisch eine Lizenz für alle Patente, die er an diesem Code hält.
* Das schützt die Nutzer vor "Patent-Trollen". Cuando se trata de una patente de patente, der selbst am Projekt mitgearbeitet hat, verliert er automatisch seine Rechte an der Software.

### Atención al cliente:
Bajo **Apache 2.0**, señalización profesional:

1. **Rechtssicherheit:** Besonders für große Firmen (como en Automoción o Finanzas) ist das die "sicherste" Lizenz.
2. **Flexibilidad:** Puede integrar los problemas de herramientas en sus cadenas de herramientas internas, sin tener que preocuparse por el efecto Copyleft.

**Fazit:** Es FOSS (software gratuito y de código abierto) lo mejor que puedo hacer. Es ist "frei" sowohl im Sinne von "kostenlos" als auch im Sinne von "Freiheit".
