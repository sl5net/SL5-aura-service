# Deutscher Sprache의 Die License erklärt

**Apache 라이센스, 버전 2.0**은 **자유 소프트웨어**(무료 소프트웨어) 및 **오픈 소스 소프트웨어**로 확정되었습니다.

Hier sind die Details, warum das so ist und was das für dich bedeutet:

### 1. 오피지엘 아네르케눙
* **FSF(자유 소프트웨어 재단):** "자유 소프트웨어"를 위한 Lizenz와 마찬가지로 Apache 2.0도 포함됩니다. 이 알레르기는 **nicht kompatibel mit der GPL Version 2**, aber kompatibel mit der **GPL Version 3**입니다.
* **OSI(오픈 소스 이니셔티브):** Apache 2.0이 대중화되고 표준화되어 오픈 소스 라이선스로 변경되었습니다.

### 2. 다프맨은 다미트 메이헨이었나요? (다이 프라이헤이텐)
Unter der Apache 2.0 Lizenz darf jeder(Firmen 제공):
* **Die Software for jeden Zweck nutzen:** Kommerziell, privat, intern – alles erlaubt.
* **Den Code verändern:** Du kannst das Tool anpassen, wie du willst.
* **Die Software kopieren und verbreiten:** Du darfst sie weitergeben.
* **Sogar verkaufen:** Du darfst Software, die auf Apache-Code basiert, verkaufen.

### 3. Besonderheit: "허용적"(Freizügig)
Im Gegensatz zur GPL(General Public License)은 Apache-Lizenz **keinen "Copyleft"-Effekt**를 사용합니다.
* 기본 사항: Wenn du Aura(Apache 2.0 이하) nimmst und veränderst, **musst** du deinen neuen Code nicht zwingend wieder als Open Source veröffentlichen. Du kannst ihn auch in ein geschlossenes, kommerzielles Produkt einbauen.

### 4. Ein großer Vorteil: Die Patentklausel
Apache 2.0 Lizenz ist bei Firmen sehr beliebt(종종 MIT 또는 BSD와 같은 신자), weil sie eine **ausdrückliche Patenterteilung** enthält.
* Jeder, der Code zum Projekt beiträgt, gewährt den Nutzern automatisch eine Lizenz für alle Patente, die er an diesem Code hält.
* Das schützt die Nutzer vor "Patent-Trollen". Wenn jemand dich wegen Patentverletzung verklagt, der selbst am Projekt mitgearbeitet hat, verliert er automatisch seine Rechte an der Software.

### 주삼멘파숭:
**Apache 2.0**의 경우 가장 전문적인 신호는 다음과 같습니다.

1. **Rechtssicherheit:** Besonders für große Firmen(자동차 산업 금융 분야) ist das die "sicherste" Lizenz.
2. **유연함:** 내부 도구 체인 통합에 도구 문제가 있으면 Copyleft-Effekt haben zu müssen에 대해 걱정해야 합니다.

**Fazit:** FOSS(무료 및 오픈 소스 소프트웨어)는 Sinne을 대표합니다. Es ist "frei" sowohl im Sinne von "kostenlos" as auch im Sinne von "Freiheit".
