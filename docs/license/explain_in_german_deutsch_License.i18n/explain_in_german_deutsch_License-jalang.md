# ドイツ語でのライセンス erklärt

**Apache ライセンス、バージョン 2.0** は、**フリー ソフトウェア** (Freie Software) であり、**オープン ソース ソフトウェア** であると明確です。

詳細を参照してください。これは、実際に行われたことです。

### 1. オフィツィエル・アネルケヌング
* **FSF (Free Software Foundation):** Apache 2.0 は「Freie Software」用に開発されました。 **GPL バージョン 2** と、**GPL バージョン 3** に関する注意事項があります。
* **OSI (Open Source Initiative):** Apache 2.0 は、オープンソースのライブラリと標準化されています。

### 2. ダーフ男はダミット・マッチェンでしたか? (『フライハイテン』)
Apache 2.0 のバージョン (auch Firmen):
* **Die Software für jeden Zweck Nutzen:** Kommerziell、プライベート、インターン – alles erlaubt。
* **コードバージョン:** ツールを使用して、意志を表示します。
* **ソフトウェアのコピーと詳細:** 情報を参照してください。
* **Sogar verkaufen:** Apache-Code ベースのソフトウェアを使用してください。

### 3. Die Besonderheit: 「寛容」 (Freizügig)
Im Gegensatz zur GPL (General Public License) は、Apache-Lizenz **keinen "Copyleft"-Effekt** に準拠しています。
* 推奨事項: Wenn du Aura (Apache 2.0 以降) の最新情報、**musst** du deinen neuen Code nicht zwingend wieder als Open Source veröffentlichen。 Du kannst ihn auch in ein geschlossenes、kommerzielles Produkt einbauen。

### 4. アイン・グローサー・ヴォルテイル: パテントクラウゼルの死
Apache 2.0 は完全に信頼されており (BSD も MIT も信じています)、**ausdrückliche Patenterreilung** を重視しています。
* Jeder、der Code zum Projekt beiträgt、gewährt den Nutzern automatisch eine Lizenz für alle Patente、die er an diesem Code halt。
* 「Patent-Trollen」の記録。ソフトウェアを自動化するために、特許を取得したり、プロジェクトを承認したりすることができます。

### ズサンメンファッスン:
**Apache 2.0** よりも、最もプロフェッショナルな情報を提供します:

1. **Rechtssicherheit:** Besonders für große Firmen (wie in Automotive or or Finance) ist das die "sicherste" Lizenz。
2. **柔軟性:** 内部ツールチェーンの統合におけるツールの問題については、コピーレフトの影響を考慮する必要があります。

**ファジット:** 私は FOSS (フリーおよびオープン ソース ソフトウェア) を最も大切にしています。 Es ist "frei" sowohl im Sinne von "kostenlos" als auch im Sinne von "Freiheit"。
