# 德语版 Die License erklärt

**Apache 许可证，版本 2.0** ist definitiv **自由软件**（自由软件）以及**开源软件**。

Hier sind die 详细信息，warum das so ist und was das für dich bedeutet：

### 1. 办公室
* **FSF（自由软件基金会）：** Apache 2.0 是 Lizenz 的“自由软件”。请注意，**不支持 GPL 版本 2**，不支持 **GPL 版本 3**。
* **OSI（开源倡议）：** Sie erkennt die Apache 2.0 是流行的和标准化的开源 Lizenzen 和。

### 2. darf man damit machen 是吗？ （自由）
在 Apache 2.0 Lizenz darf jeder (auch Firmen) 下：
* **Die Software für jeden Zweck nutzen：** Kommerziell，私人，实习生 – alles erlaubt。
* **Den Code verändern：** Du kannst das Tool anpassen，wie du willst。
* **Die Software kopieren und verbreiten：** Du darfst sie weitergeben。
* **Sogar verkaufen：** Du darfst Software，die auf Apache-Code basiert，verkaufen。

### 3. Die Besonderheit：“宽容”（Freizügig）
Im Gegensatz zur GPL（通用公共许可证），是 Apache-Lizenz **keinen“Copyleft”-Effekt**。
* 说明：Wenn du Aura（Apache 2.0 下）nimmst und veränderst，**必须** du deinen neuen 代码 nicht zwingend wieder als Open Source veröffentlichen。您可以在整个过程中观察到产品的变化。

### 4. Ein großer Vorteil：Die Patentklausel
Apache 2.0 Lizenz 是坚定的信念（通常是 MIT 或 BSD 的信念），我们对此充满热情。
* Jeder, der Code zum Projekt beiträgt, gewährt den Nutzern automatisch eine Lizenz für alle Patente, die er an diesem Code hält.
* Das schützt die Nutzer vor“Patent-Trollen”。如果我们已经获得了专利权，那么我们就可以自行完成项目，并自动获得软件专利权。


在 **Apache 2.0** 下，最专业：

1. **Rechtssicherheit：** Besonders für große Firmen（即汽车或金融）是利森茨的“sicherste”。
2. **灵活性：** Sie können dein 工具问题在内部工具链集成中，ohne Angst vor dem Copyleft-Effekt haben zu müssen。

**Fazit：** Es ist FOSS（自由开源软件）im besten Sinne。 Es ist "frei" sowohl im Sinne von "kostenlos" als auch im Sinne von "Freiheit"。
