# Aura 管理 UI

管理 UI を使用すると、ブラウザで Aura 設定を表示および変更できます。

＃＃ 開ける

http://ローカルホスト:8084

## あなたにできること

- 各インターフェース（音声、端末、ウェブ）の翻訳状況を確認
- インターフェースごとに変換を有効または無効にします
- ターゲット言語を選択します (英語、フランス語、スペイン語など)

## インターフェース

|インターフェース |説明 |
|----------|-------------------------------------|
|スピーチ |音声入力（マイク） |
|ターミナル |コマンドライン (`s` コマンド) |
|ウェブ | Streamlit Web チャット (ポート 8831) |

＃＃ 例

Web ユーザーのみを英語に翻訳するには、音声と端末をオフのままにします。
言語「en」で Web を有効にします。
