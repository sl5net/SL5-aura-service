# Aura 管理 UI

管理 UI を使用すると、アイドル状態のリソースのコストをゼロにしながら、ブラウザで Aura 設定を表示および変更できます。ダッシュボード サーバーは起動時に実行されません。要求された場合にのみオンデマンドで開始されます。

## 開く方法 (オンデマンド)

次の 3 つの方法のいずれかを使用して、管理者ダッシュボードを動的に起動して開くことができます。

### 1. 音声コマンド
マイクに向かって話すだけです。
* *「オーラ管理」

### 2. ターミナル/コンソールコマンド
ターミナルで作業している場合は、次のコマンドを実行してランチャーを直接トリガーします。
```bash
s aura administration
```

*⚠️ **Windows/macOS ユーザー向けのプラットフォームに関する注意事項:** 短い「s」コマンド ラッパーは主に Linux 環境用に構成されています。それについてはドキュメントをお読みください。 Windows または macOS を実行している場合、「s」コマンドはそのままでは機能しない可能性があります。オペレーティング システムの「s」コマンド エイリアスを設定および実装する方法については、公式 CLI セットアップ ドキュメントを参照してください。*


### 3. デスクトップのショートカット
プラットフォーム固有のデスクトップ アイコンを作成するには、次のセットアップ スクリプトを 1 回実行します。
```bash
python scripts/py/chat/install_shortcut.py
```
次に、デスクトップ上の **Aura Admin Dashboard** アイコンをダブルクリックします。

## ブラウザからの直接アクセス
上記のオンデマンド方法のいずれかを介してサーバーが起動されると、いつでもブラウザーでインターフェイスに直接アクセスできます。

http://ローカルホスト:8084

*(このリンクをブラウザでブックマークしてください!)*

---

## あなたにできること

- 各インターフェース（音声、端末、Web）の翻訳状況を確認します。
- インターフェースごとに変換を有効または無効にします。
- ターゲット言語を選択します (英語、フランス語、スペイン語など)。

## インターフェース

|インターフェース |説明 |
|----------|-------------------------------------|
|スピーチ |音声入力（マイク） |
|ターミナル |コマンドライン (`s` コマンド) |
|ウェブ | Streamlit Web チャット (ポート 8831) |

＃＃ 例

Web ユーザーのみを英語に翻訳するには、音声と端末をオフのままにし、言語「en」で Web を有効にします。
