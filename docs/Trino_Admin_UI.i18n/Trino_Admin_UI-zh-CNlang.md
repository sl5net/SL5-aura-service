# 光环管理界面

管理 UI 允许您在浏览器中查看和更改 Aura 设置。

＃＃ 打开

http://本地主机:8084

## 你可以做什么

- 查看每个界面（语音、终端、Web）的翻译状态
- 启用或禁用每个接口的转换
- 选择目标语言（英语、法语、西班牙语等）

## 接口

|接口 |描述 |
|------------------------|------------------------------------|
|演讲|语音输入（麦克风）|
|终端|命令行（`s`命令）|
|网络| Streamlit 网络聊天（端口 8831）|

＃＃ 例子

仅将网络用户翻译成英语 - 关闭语音和终端，
使用语言“en”启用网络。
