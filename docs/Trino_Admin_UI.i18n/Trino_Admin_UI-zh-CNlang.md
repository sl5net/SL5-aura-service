# 光环管理界面

管理 UI 允许您在浏览器中查看和更改 Aura 设置，且闲置资源成本为零。仪表板服务器不会在启动时运行；仅在请求时才按需启动。

## 如何打开（按需）

您可以使用以下三种方法之一动态启动并打开管理仪表板：

### 1. 语音命令
只需对着麦克风讲话：
**“光环管理”

### 2. 终端/控制台命令
如果您在终端中工作，请运行以下命令直接触发启动器：
__代码_块_0__

*⚠️ **针对 Windows/macOS 用户的平台说明：** 短 `s` 命令包装器主要针对 Linux 环境进行配置。请阅读文档。如果您运行的是 Windows 或 macOS，“s”命令可能无法开箱即用。请参阅我们的官方 CLI 设置文档，了解如何为您的操作系统配置和实现 `s` 命令别名。*


### 3.桌面快捷方式
要创建特定于平台的桌面图标，请运行此安装脚本一次：
__代码_块_1__
然后，只需双击桌面上的 **Aura 管理仪表板** 图标即可。

---

## 直接浏览器访问
通过上述任何一种按需方法启动服务器后，您可以随时直接在浏览器中访问该界面：

http://本地主机:8084

*（请随时在浏览器中将此链接添加为书签！）*

---

## 你可以做什么

- 查看每个界面（语音、终端、Web）的翻译状态。
- 启用或禁用每个接口的转换。
- 选择目标语言（英语、法语、西班牙语等）。

## 接口

|接口 |描述 |
|------------------------|------------------------------------|
|演讲|语音输入（麦克风）|
|终端|命令行（`s`命令）|
|网络| Streamlit 网络聊天（端口 8831）|

＃＃ 例子

要仅将网络用户翻译成英语 - 关闭语音和终端，启用语言为“en”的网络。
