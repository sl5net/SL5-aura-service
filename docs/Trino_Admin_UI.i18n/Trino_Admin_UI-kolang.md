# 아우라 관리 UI

관리 UI를 사용하면 유휴 리소스 비용 없이 브라우저에서 Aura 설정을 보고 변경할 수 있습니다. 대시보드 서버는 부팅 시 실행되지 않습니다. 요청된 경우에만 온디맨드 방식으로 시작됩니다.

## 개설 방법(주문형)

다음 세 가지 방법 중 하나를 사용하여 관리 대시보드를 동적으로 시작하고 열 수 있습니다.

### 1. 음성 명령
마이크에 대고 말하세요.
* *"오라 행정"

### 2. 터미널/콘솔 명령
터미널에서 작업하는 경우 다음 명령을 실행하여 실행 프로그램을 직접 트리거하십시오.
```bash
s aura administration
```

*⚠️ **Windows/macOS 사용자를 위한 플랫폼 참고 사항:** 짧은 `s` 명령 래퍼는 주로 Linux 환경용으로 구성됩니다. 이에 대한 문서를 읽어보십시오. Windows 또는 macOS를 실행하는 경우 `s` 명령이 기본적으로 작동하지 않을 수 있습니다. 운영 체제에 대한 `s` 명령 별칭을 구성하고 구현하는 방법을 알아보려면 공식 CLI 설정 문서를 참조하세요.*


### 3. 바탕화면 바로가기
플랫폼별 데스크탑 아이콘을 생성하려면 다음 설정 스크립트를 한 번 실행하십시오.
```bash
python scripts/py/chat/install_shortcut.py
```
그런 다음 데스크탑에서 **Aura Admin Dashboard** 아이콘을 두 번 클릭하기만 하면 됩니다.

## 직접 브라우저 액세스
위의 주문형 방법 중 하나를 통해 서버가 실행되면 언제든지 브라우저에서 직접 인터페이스에 액세스할 수 있습니다.

http://localhost:8084

*(브라우저에서 이 링크를 즐겨찾기에 추가하세요!)*

---

## 당신이 할 수 있는 일

- 각 인터페이스(음성, 터미널, 웹)의 번역 상태를 확인하세요.
- 인터페이스별로 번역을 활성화 또는 비활성화합니다.
- 대상 언어(영어, 프랑스어, 스페인어 등)를 선택하세요.

## 인터페이스

| 인터페이스 | 설명 |
|------------|----------|
| 연설 | 음성입력(마이크) |
| 터미널 | 명령줄(`s` 명령) |
| 웹 | 간소화된 웹 채팅(포트 8831) |

## 예

웹 사용자만 영어로 번역하려면 음성 및 터미널을 꺼두고 'en' 언어로 웹을 활성화하세요.
