# 아우라 관리 UI

관리 UI를 사용하면 브라우저에서 Aura 설정을 보고 변경할 수 있습니다.

## 열려 있는

http://localhost:8084

## 당신이 할 수 있는 일

- 각 인터페이스(음성, 터미널, 웹)의 번역 상태를 확인하세요.
- 인터페이스별 번역 활성화 또는 비활성화
- 대상 언어 선택(영어, 프랑스어, 스페인어 등)

## 인터페이스

| 인터페이스 | 설명 |
|------------|----------|
| 연설 | 음성입력(마이크) |
| 터미널 | 명령줄(`s` 명령) |
| 웹 | 간소화된 웹 채팅(포트 8831) |

## 예

웹 사용자만 영어로 번역하려면 음성 및 터미널을 꺼두고,
'en' 언어로 웹을 활성화합니다.
