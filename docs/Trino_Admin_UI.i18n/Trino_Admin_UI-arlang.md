# واجهة المستخدم الإدارية Aura

تتيح لك واجهة المستخدم الإدارية عرض إعدادات Aura وتغييرها في متصفحك.

                                                                  ## يفتح

                                                        http://localhost:8084

                                                  ## ما يمكنك فعله

- انظر حالة الترجمة لكل واجهة (الكلام، المحطة، الويب)
                - تمكين أو تعطيل الترجمة لكل واجهة
- اختر اللغة المستهدفة (الإنجليزية والفرنسية والإسبانية وغيرها)

                                                              ## واجهات

                                              | الواجهة | الوصف |
                           |-----------|------------------------------------|
                      | خطاب | إدخال الصوت (ميكروفون) |
                            | محطة | سطر الأوامر (أمر `s`) |
         | الويب | دردشة الويب Streamlit (المنفذ 8831) |

                                                                  ## مثال

لترجمة مستخدمي الويب فقط إلى اللغة الإنجليزية - اترك الكلام والمحطة الطرفية مغلقة،
                                     تمكين الويب باللغة `en`.
