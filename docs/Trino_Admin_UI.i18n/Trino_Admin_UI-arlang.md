# واجهة المستخدم الإدارية Aura

تتيح لك واجهة المستخدم الإدارية عرض إعدادات Aura وتغييرها في متصفحك دون أي تكلفة للموارد الخاملة. خادم لوحة المعلومات لا يعمل عند التمهيد؛ يتم تشغيله عند الطلب فقط عند الطلب.

                                 ## كيفية الفتح (عند الطلب)

يمكنك تشغيل لوحة تحكم المشرف وفتحها ديناميكيًا باستخدام أي من الطرق الثلاث التالية:

                                         ### 1. الأوامر الصوتية
               ما عليك سوى التحدث عبر الميكروفون:
                                                 * *"إدارة الهالة"

              ### 2. أمر الوحدة الطرفية/وحدة التحكم
إذا كنت تعمل في الوحدة الطرفية، فقم بتشغيل هذا الأمر لتشغيل المشغل مباشرةً:
```bash
s aura administration
```

*⚠️ **ملاحظة النظام الأساسي لمستخدمي Windows/macOS:** تم تكوين مجمّع الأوامر القصير بشكل أساسي لبيئات Linux. يرجى قراءة الوثيقة لذلك. إذا كنت تستخدم نظام التشغيل Windows أو macOS، فقد لا يعمل الأمر s خارج الصندوق. يرجى الرجوع إلى وثائق إعداد CLI الرسمية الخاصة بنا لمعرفة كيفية تكوين وتنفيذ الاسم المستعار للأمر s لنظام التشغيل الخاص بك.*


                                      ### 3. اختصار سطح المكتب
لإنشاء رمز سطح مكتب خاص بالنظام الأساسي، قم بتشغيل البرنامج النصي للإعداد مرة واحدة:
```bash
python scripts/py/chat/install_shortcut.py
```
بعد ذلك، ما عليك سوى النقر نقرًا مزدوجًا فوق أيقونة **Aura Admin Dashboard** الموجودة على سطح المكتب لديك.

                                ## الوصول المباشر للمتصفح
بمجرد تشغيل الخادم عبر أي من الطرق حسب الطلب المذكورة أعلاه، يمكنك الوصول إلى الواجهة مباشرة في متصفحك في أي وقت:

                                                        http://localhost:8084

*(لا تتردد في وضع إشارة مرجعية على هذا الرابط في متصفحك!)*

                                                                          ---

                                                  ## ما يمكنك فعله

- انظر حالة الترجمة لكل واجهة (الكلام، المحطة، الويب).
               - تمكين أو تعطيل الترجمة لكل واجهة.
- اختر اللغة المستهدفة (الإنجليزية والفرنسية والإسبانية وغيرها).

                                                              ## واجهات

                                              | الواجهة | الوصف |
                           |-----------|------------------------------------|
                      | خطاب | إدخال الصوت (ميكروفون) |
                            | محطة | سطر الأوامر (أمر `s`) |
         | الويب | دردشة الويب Streamlit (المنفذ 8831) |

                                                                  ## مثال

لترجمة مستخدمي الويب فقط إلى الإنجليزية - اترك الكلام والمحطة الطرفية قيد الإيقاف، وقم بتمكين الويب باستخدام اللغة `en`.
