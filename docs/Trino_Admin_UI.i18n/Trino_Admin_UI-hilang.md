# ऑरा एडमिन यूआई

एडमिन यूआई आपको अपने ब्राउज़र में ऑरा सेटिंग्स देखने और बदलने की सुविधा देता है।

## खुला

http://localhost:8084

## आप क्या कर सकते हैं

- प्रत्येक इंटरफ़ेस (भाषण, टर्मिनल, वेब) के लिए अनुवाद स्थिति देखें
- प्रति इंटरफ़ेस अनुवाद सक्षम या अक्षम करें
- लक्ष्य भाषा चुनें (अंग्रेजी, फ्रेंच, स्पेनिश, आदि)

## इंटरफ़ेस

| इंटरफ़ेस | विवरण |
|--------|------------------------------------------------|
| भाषण | वॉयस इनपुट (माइक्रोफोन) |
| टर्मिनल | कमांड लाइन (`s` कमांड) |
| वेब | स्ट्रीमलिट वेब चैट (पोर्ट 8831) |

## उदाहरण

केवल वेब उपयोगकर्ताओं का अंग्रेजी में अनुवाद करने के लिए - भाषण और टर्मिनल को बंद रखें,
भाषा `एन` के साथ वेब सक्षम करें।
