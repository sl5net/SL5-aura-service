## 阈值指导

|场景 |建议的“wer_threshold” |
|------------------------------------------|--------------------------|
|洁净工作室演讲，英语 | 0.10 – 0.15 |
|休闲演讲，英语 | 0.20 – 0.30 |
|德语演讲 | 0.15 – 0.25 |
|嘈杂的背景| 0.30 – 0.45 |

开始放松 (0.35)，并在看到 Vosk 实际产生的效果后收紧。

## 为什么 YouTube 文字记录被视为基本事实？

YouTube 的自动字幕并不完美，但它们是：
- **始终可用**您自己的视频
- **免费** – 无需手动标记
- **足够好**来捕获严重的回归
- 由不同的（Google）ASR引擎生成→独立参考

WER 比较捕获了代码更改导致 Vosk 的回归
真实音频的效果明显较差，无需手动转录
任何事物。
