## यूट्यूब ट्रांस्क्रिप्ट को जमीनी सच्चाई क्यों मानते हैं?

यूट्यूब के ऑटो-कैप्शन सही नहीं हैं, लेकिन वे हैं:
- **आपके अपने वीडियो के लिए हमेशा उपलब्ध**
- **निःशुल्क** - किसी मैन्युअल लेबलिंग की आवश्यकता नहीं है
- **गंभीर प्रतिगमन को पकड़ने के लिए काफी अच्छा**
- एक अलग (Google) ASR इंजन द्वारा निर्मित → स्वतंत्र संदर्भ

WER तुलना प्रतिगमन को पकड़ती है जहां एक कोड परिवर्तन वोस्क बनाता है
मैन्युअल रूप से ट्रांसक्राइब करने की आवश्यकता के बिना, वास्तविक ऑडियो पर काफी खराब
कुछ भी।
